N2 Search
=========

`N2 Search <https://www.n2sm.net/n2search.html>`__ differs from Fess in the following ways:

- Improved search performance

- Optimized disk usage

- Enhanced security for administrative features

- Morphological analysis supporting new words

Support Services
================

If you need the following services, please consider `Commercial Support (Search Solution Support) <https://www.n2sm.net/products/n2search/support.html>`__:

- Deployment and operation consulting

- Custom development for specific requirements (The Fess project addresses general requirements but does not support individual-specific requirements)

- Large-scale document crawling (Projects involving more than 500,000 documents require sufficient knowledge of Fess/OpenSearch)

- Cluster configuration

- Search result optimization for enterprise/internal search

- Fess/OpenSearch server tuning

- Investigation and support for files that cannot be indexed (The Fess project does not support cases where dependent libraries cannot read files; please consult commercial support if needed)

- High-speed full-text search system environment construction

- Maintenance of specific versions (The Fess project only provides fixes for the latest version)

- Training

- Other services

Frequently Asked Questions
==========================

Q: What is the difference between the open source and commercial editions?
--------------------------------------------------------------------------

A: The open source edition (Fess) is free to use, but technical support is community-based.
The commercial edition (N2 Search) includes improved search performance, enhanced security features, and professional support.

Q: Can I try it before purchasing?
-----------------------------------

A: Yes. The open source Fess can be downloaded and tried immediately from the `download page <downloads.html>`__.
For N2 Search, please contact us via the inquiry form to discuss a trial.

Q: Can I migrate from Fess to N2 Search?
------------------------------------------

A: Yes, migration is possible. For detailed migration procedures, please contact commercial support.
We will guide you on index data migration and configuration transfer.

Q: Are support contracts annual?
---------------------------------

A: Support contract terms are flexible and can be tailored to your needs.
For details, please contact us via the `N2SM contact form <https://www.n2sm.net/contact.html>`__.

Q: Can Fess be used in cloud environments?
-------------------------------------------

A: Yes, it can be used on major cloud platforms such as AWS, Azure, and GCP.
For Docker-based deployment, see the `Docker Installation Guide <15.5/install/install-docker.html>`__.
