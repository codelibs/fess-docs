============
Commercial Support
============

About Commercial Support
========================

Fess is open source software provided under the Apache License and can be used freely for personal or commercial purposes.

`N2SM, Inc. <https://www.n2sm.net/>`__ provides:

- `N2 Search <https://www.n2sm.net/n2search.html>`__: Commercial product version of Fess

- `Search Solution Support <https://www.n2sm.net/products/n2search/support.html>`__: Customization and deployment assistance for Fess and Elasticsearch

N2 Search
=========

`N2 Search <https://www.n2sm.net/n2search.html>`__ differs from Fess in the following ways:

- Improved search performance

- Optimized disk usage

- Enhanced security for administrative features

- Morphological analysis supporting new words

Support Services
================

If you need the following services, please consider `Commercial Support (Search Solution Support) <https://www.n2sm.net/products/n2search/support.html>`__:

- Deployment and operation consulting

- Custom development for specific requirements (The Fess project addresses general requirements but does not support individual-specific requirements)

- Large-scale document crawling (Projects involving more than 500,000 documents require sufficient knowledge of Fess/Elasticsearch)

- Cluster configuration

- Search result optimization for enterprise/internal search

- Fess/Elasticsearch server tuning

- Investigation and support for files that cannot be indexed (The Fess project does not support cases where dependent libraries cannot read files; please consult commercial support if needed)

- High-speed full-text search system environment construction

- Maintenance of specific versions (The Fess project only provides fixes for the latest version)

- Training

- Other services

AWS Marketplace
===============

With feature limitations such as document count caps, N2 Search is available on AWS Marketplace.
It can be easily deployed as an enterprise document search engine for small and medium-sized businesses.
See `N2 Search <https://aws.amazon.com/marketplace/pp/B014JFU5EW>`__.
