======
시작하기
======

본 문서에 대하여
==================

이 설치 가이드는 |Fess| 의 설치, 시작, 업그레이드 및 제거 절차를 설명합니다.
사용자의 환경에 맞는 적절한 설치 방법을 선택하시기 바랍니다.

대상 독자
========

이 문서는 |Fess| 의 설치 작업 및 제거 작업을 담당하는 사용자를 대상으로 합니다.

읽기 전에
==============

이 문서에서는 |Fess| 의 설치 방법을 안내합니다. 다음의 지식이 필요합니다:

- 서버 관리 기본 지식
- Linux 또는 Windows 명령줄 작업
- 네트워크 및 방화벽 기본 설정
- Docker 기본 지식 (Docker 버전 사용 시)

문서 구성
==============

이 설치 가이드는 다음 섹션으로 구성되어 있습니다:

- **시스템 요구사항**: |Fess| 를 실행하는 데 필요한 하드웨어 및 소프트웨어 요구사항
- **설치**: 플랫폼별 설치 절차

  - Linux (TAR.GZ/RPM/DEB) 설치
  - Windows (ZIP) 설치
  - Docker 설치

- **시작 및 중지**: |Fess| 서버의 시작, 중지 및 초기 설정
- **업그레이드**: 이전 버전에서 최신 버전으로 업그레이드하는 절차
- **제거**: |Fess| 의 완전한 제거 절차
- **보안 설정**: 운영 환경에서 권장되는 보안 설정
- **문제 해결**: 일반적인 문제 및 해결 방법
