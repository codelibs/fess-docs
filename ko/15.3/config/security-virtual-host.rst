========
가상 호스트
========

가상 호스트에 대하여
==============

|Fess| 에 접근했을 때의 호스트명(URL의 호스트 부분)에 따라 검색 결과를 구분할 수 있습니다.
검색 결과는 각각의 JSP에서 표시되므로 디자인을 변경할 수도 있습니다.

시스템 설정
----------

:doc:`관리자 가이드 > 전반 설정 <../admin/general-guide>` 에서 "가상 호스트"를 설정합니다. 여기서 설정한 가상 호스트명을 크롤 설정에서 지정합니다.

**형식**

::

   Host:호스트명[:포트번호]=가상호스트명

.. list-table::

   * - *호스트명*
     - 시스템에서 이름 확인(DNS)이 가능한 호스트명 또는 IP 주소
   * - *포트번호*
     - 옵션. 기본값은 80.
   * - *가상호스트명*
     - 크롤 설정에서 지정하는 가상 호스트명

**예**

::

   Host:abc.example.com:8080=host1
   Host:192.168.1.123:8080=host2

설정을 수행하면 ``WEB-INF/view/가상호스트명`` 에 검색 페이지의 JSP가 생성됩니다.
이것들을 편집하여 가상 호스트마다 페이지 디자인을 변경할 수도 있습니다.


크롤 설정
---------

웹 크롤 설정, 파일 크롤 설정, 데이터 스토어 크롤 설정 중 하나에서 "가상 호스트"를 지정합니다.
"가상 호스트"는 시스템 설정에서 설정한 가상 호스트명 중 하나를 지정합니다.

**예**

.. list-table::

   * - *가상 호스트*
     - ``host1``
