======
OR 검색
======

검색어 중 하나라도 포함된 문서를 검색하려는 경우 OR 검색을 사용합니다. 검색어 입력란에 여러 단어를 입력한 경우 기본적으로 AND 검색이 수행됩니다.

사용 방법
------

OR 검색을 사용하는 경우 검색어 사이에 OR을 입력합니다. OR은
반각 대문자로 입력해야 하며, 앞뒤로 공백이 필요합니다.

예를 들어, "검색어1"과 "검색어2" 중 하나라도 포함된 문서를 검색하려는 경우 다음과 같이 검색 양식에 입력합니다.

::

    검색어1 OR 검색어2

OR로 여러 단어를 연결할 수도 있습니다.
