======
상세 검색
======

상세 검색 화면에서는 더욱 복잡한 검색을 수행할 수 있습니다.

사용 방법
------

상세 검색 화면은 검색 화면의 옵션에 있는 상세 검색 버튼에서 접속할 수 있습니다.

|image0|

페이지 하단의 검색 버튼을 클릭하여 검색할 수 있습니다.

항목 목록
------

모든 단어 포함
::::::::::::::

입력한 모든 단어를 포함하는 문서를 검색할 수 있습니다.
단어는 공백으로 구분됩니다.

어순 포함 완전 일치
:::::::::::::::

입력한 문자열을 포함하는 문서를 검색할 수 있습니다.
공백 등으로 구분할 수 없습니다.

단어 중 하나라도 포함
:::::::::::::::

입력한 단어 중 하나라도 포함하는 문서를 검색할 수 있습니다.
단어는 공백으로 구분됩니다.

제외할 단어
:::::::::

입력한 모든 단어를 포함하지 않는 문서를 검색할 수 있습니다.
단어는 공백으로 구분됩니다.

표시 건수
::::::

검색 결과의 한 페이지 내에 포함되는 검색 결과 건수를 지정할 수 있습니다.

정렬
:::::

검색 일시 등의 필드를 지정하여 검색 결과를 정렬할 수 있습니다.

우선 언어
:::::::::

검색 결과에서 우선할 언어를 지정할 수 있습니다.

라벨
:::::

라벨 정보로 필터링 검색할 수 있습니다. 라벨이 등록되지 않은 경우 표시되지 않습니다.

업데이트 일시
::::::

문서의 업데이트 일시로 필터링 검색할 수 있습니다.

파일 형식
:::::::::

문서의 파일 형식으로 필터링 검색할 수 있습니다.

검색 대상
::::::

검색 대상을 페이지 전체, 페이지 제목, 페이지 URL 중에서 지정할 수 있습니다.

사이트 또는 도메인
:::::::::::::::

입력한 사이트 또는 도메인으로 필터링 검색할 수 있습니다.
