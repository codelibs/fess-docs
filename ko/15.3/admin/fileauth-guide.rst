=========
파일 인증
=========

개요
====

여기서는 파일을 대상으로 한 크롤링에서 파일 인증이 필요한 경우의 설정 방법에 대해 설명합니다.
|Fess| 는 FTP 또는 Windows의 공유 폴더에 대한 인증을 지원합니다.

관리 방법
======

표시 방법
------

아래 그림의 파일 인증 설정 목록 페이지를 열려면 왼쪽 메뉴의 [크롤러 > 파일 인증]을 클릭합니다.

|image0|

편집하려면 설정 이름을 클릭합니다.

설정 생성
--------

파일 인증 설정 페이지를 열려면 신규 생성 버튼을 클릭합니다.

|image1|

설정 항목
------

호스트 이름
::::::

인증이 필요한 사이트의 호스트 이름을 지정합니다.

포트
::::

인증이 필요한 사이트의 포트를 지정합니다.

스키마
::::::

인증 방법을 선택합니다.
FTP 또는 SAMBA (Windows 공유 폴더 인증)를 사용할 수 있습니다.

사용자 이름
::::::::

인증 사이트에 로그인하기 위한 사용자 이름을 지정합니다.

비밀번호
::::::::

인증 사이트에 로그인하기 위한 비밀번호를 지정합니다.

매개변수
:::::::::

인증 사이트에 로그인하는 데 필요한 설정 값이 있는 경우 설정합니다. SAMBA의 경우 domain 값을 설정할 수 있습니다. 설정하는 경우 다음과 같이 기술합니다.

::

    domain=FUGA

파일 크롤링 설정
:::::::::::::::

이 인증 설정을 사용할 크롤링 설정을 지정합니다.

설정 삭제
--------

목록 페이지의 설정 이름을 클릭하고 삭제 버튼을 클릭하면 확인 화면이 표시됩니다.
삭제 버튼을 누르면 설정이 삭제됩니다.
