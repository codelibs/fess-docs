=========
부스트 검색
=========

부스트 검색(가중치 검색)
====================

검색어 중 특정 검색어를 우선하려는 경우 부스트 검색을 사용합니다. 부스트 검색을 사용하면 검색어의 중요도에 따른 검색이 가능합니다.

사용 방법
------

부스트 검색을 사용하려면 검색어 뒤에 "^부스트값" 형식으로 부스트 값(가중치 값)을 지정합니다.

예를 들어, "사과 귤"을 검색하려는 경우 "사과"가 더 많이 포함된 페이지를 검색하려면 다음과 같이 검색 양식에 입력합니다.

::

    사과^100 귤

부스트 값은 1 이상의 정수를 지정합니다.
