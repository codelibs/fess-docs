=========
위치 정보 검색
=========

인덱스 생성 시 각 문서에 대해 위도 경도의 위치 정보를 추가함으로써 검색 시 위치 정보를 사용한 검색이 가능합니다.

사용 방법
------

기본적으로 다음 매개변수를 사용할 수 있습니다.

.. list-table::

   * - geo.<fieldname>.point
     - 위도, 경도의 도분초를 Double 형식으로 지정합니다.
   * - geo.<fieldname>.distance
     - 문서와의 거리를 킬로미터 단위로 지정합니다. 예: 10km

표: 요청 매개변수
