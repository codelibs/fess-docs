========
역할 검색
========

|Fess| 는 사용자 관리 기능을 제공하며 사용자가 로그인하면 역할 검색을 사용할 수 있습니다. |Fess| 에서 관리하는 사용자는 로그인 후 역할 검색 및 사용자 비밀번호를 변경할 수 있습니다.


검색 방법
------

역할을 설정하고 크롤링 및 인덱싱한 상태라면 검색 결과는 역할을 보유한 사용자에게만 표시할 수 있습니다.
사용자가 로그인한 상태라면 해당 사용자가 속한 역할에 따라 검색이 수행됩니다.

비밀번호 변경
------------

로그인 후 검색 화면 상단에 표시되는 사용자 이름을 클릭하면 비밀번호 변경 메뉴가 표시됩니다.

|image0|

비밀번호 변경을 클릭하면 비밀번호 변경 화면이 표시됩니다.

|image1|

현재 비밀번호와 새 비밀번호를 입력하고 업데이트 버튼을 클릭하면 비밀번호가 업데이트됩니다.
비밀번호 변경 후 뒤로 버튼을 클릭하면 검색 화면으로 돌아갈 수 있습니다.

로그아웃
--------

로그인한 상태에서 검색 화면 상단에 표시되는 사용자 이름을 클릭하고 메뉴에서 로그아웃을 선택하면 로그아웃할 수 있습니다.
