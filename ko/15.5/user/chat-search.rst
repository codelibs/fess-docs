==========================
AI 채팅 검색
==========================

개요
====

|Fess|의 AI 채팅 검색 기능을 사용하면 기존의 키워드 검색에 더해
자연스러운 대화 형식으로 정보를 찾을 수 있습니다. 질문을 입력하면
AI 어시스턴트가 검색 결과를 분석하여 알기 쉬운 응답을 생성합니다.

AI 채팅 검색의 특징
====================

대화 형식 검색
----------------

키워드를 생각하는 대신 평소 대화처럼 질문할 수 있습니다.

예:

- "Fess의 설치 방법을 알려주세요"
- "파일을 크롤링하려면 어떻게 하면 되나요?"
- "검색 결과가 표시되지 않을 때 대처 방법은?"

문맥을 이해한 응답
------------------

AI 어시스턴트는 질문의 의도를 이해하고 관련 정보를 정리하여 응답합니다.
여러 검색 결과에서 필요한 정보를 추출하여 정리된 형태로 제공합니다.

출처 명시
----------

AI의 응답에는 정보의 출처(참조 문서)가 명시됩니다.
응답의 정확성을 확인하고 싶은 경우 원본 문서를 직접 참조할 수 있습니다.

대화 계속
----------

한 번의 질문으로 끝나지 않고 대화를 계속할 수 있습니다.
이전 질문이나 응답의 문맥을 바탕으로 추가 질문을 할 수 있습니다.

채팅 검색 사용 방법
====================

채팅 시작
------------------

1. |Fess|의 검색 화면에 접속합니다
2. 화면 우측 하단의 채팅 아이콘을 클릭합니다
3. 채팅 패널이 표시됩니다

질문 입력
--------------

1. 텍스트 상자에 질문을 입력합니다
2. 전송 버튼을 클릭하거나 Enter 키를 누릅니다
3. AI 어시스턴트가 응답을 생성합니다

.. note::
   응답 생성에는 몇 초가 걸릴 수 있습니다.
   처리 중에는 현재 단계(검색 중, 분석 중 등)가 표시됩니다.

응답 확인
--------------

AI 어시스턴트의 응답이 표시됩니다. 응답에는 다음 정보가 포함됩니다:

- **응답 본문**: 질문에 대한 상세한 응답
- **출처**: 응답의 기반이 된 문서로의 링크([1], [2] 등의 형식)

출처 링크를 클릭하면 원본 문서를 확인할 수 있습니다.

대화 계속
------------

추가 질문이 있는 경우 그대로 이어서 질문할 수 있습니다:

- "좀 더 자세히 알려주세요"
- "다른 방법이 있나요?"
- "OO에 대해 자세히"

AI 어시스턴트는 이전 대화의 문맥을 고려하여 응답합니다.

새 대화 시작
------------------

다른 주제에 대해 질문하고 싶은 경우 "새 채팅" 버튼을 클릭합니다.
이렇게 하면 대화 이력이 초기화되고 새 대화를 시작할 수 있습니다.

효과적인 질문 요령
==================

구체적으로 질문하기
----------------

모호한 질문보다 구체적인 질문이 더 적절한 응답을 얻을 수 있습니다.

.. list-table::
   :header-rows: 1
   :widths: 50 50

   * - 모호한 질문
     - 구체적인 질문
   * - 설정 방법은?
     - Fess의 메모리 설정을 변경하는 방법은?
   * - 오류가 발생합니다
     - 검색 시 "인덱스를 찾을 수 없음"이라는 오류가 발생합니다
   * - 크롤링에 대해
     - 웹사이트 크롤링 시 제외 설정은 어떻게 하나요?

배경 정보 포함
----------------

상황이나 목적을 포함하면 더 적절한 응답을 얻을 수 있습니다.

좋은 예:

- "Docker 환경에서 Fess를 실행하고 있습니다. 로그 저장 위치를 변경하려면 어떻게 해야 하나요?"
- "처음 Fess를 사용합니다. 먼저 무엇을 해야 하나요?"

단계별로 질문하기
----------------

복잡한 문제는 단계별로 질문하면 이해하기 쉬워집니다.

1. "Fess에서 파일 공유를 크롤링할 수 있나요?"
2. "SMB 프로토콜로 연결하는 방법을 알려주세요"
3. "인증이 필요한 공유 폴더인 경우 어떻게 하나요?"

자주 묻는 질문
============

Q: 채팅 기능이 표시되지 않습니다
-------------------------------

A: 채팅 기능이 활성화되지 않았을 수 있습니다.
시스템 관리자에게 RAG 채팅 기능이 활성화되어 있는지 확인하세요.

Q: 응답이 표시되기까지 시간이 걸립니다
---------------------------------------

A: AI가 검색 결과를 분석하고 응답을 생성하는 데 몇 초~십여 초가 걸릴 수 있습니다.
처리 중에는 현재 단계("검색 중" "분석 중" "응답 생성 중" 등)가 표시됩니다.

Q: 응답의 출처가 잘못된 것 같습니다
-----------------------------------------

A: 출처 링크를 클릭하여 원본 문서를 확인하세요.
AI는 검색 결과를 기반으로 응답을 생성하지만, 해석에 오류가 있을 수 있습니다.
중요한 정보는 반드시 원본 문서에서 확인하는 것을 권장합니다.

Q: 이전 대화를 잊어버린 것 같습니다
-----------------------------------

A: 세션이 타임아웃되었을 수 있습니다.
일정 시간 동안 조작이 없으면 대화 이력이 초기화됩니다.
새로 대화를 시작하세요.

Q: 특정 질문에 대해 응답을 얻을 수 없습니다
---------------------------------------

A: 다음 가능성이 있습니다:

- 검색 대상 문서에 관련 정보가 없음
- 질문이 모호하여 적절한 검색이 되지 않음
- 속도 제한에 도달함

질문을 바꿔서 하거나 잠시 기다린 후 다시 시도하세요.

Q: 한국어 이외의 언어로 질문할 수 있나요?
-------------------------------

A: 네, 설정에 따라 다르지만 대부분의 경우 영어 등 다른 언어로도 질문할 수 있습니다.
AI는 질문의 언어를 인식하고 같은 언어로 응답하려고 합니다.

주의사항
========

AI 응답에 대해
----------------

- AI의 응답은 검색 결과를 기반으로 생성된 것입니다
- 응답의 정확성은 보장되지 않습니다
- 중요한 판단에는 반드시 원본 문서를 확인하세요
- 최신 정보는 공식 문서에서 확인하세요

프라이버시에 대해
--------------------

- 입력한 질문은 검색과 AI 처리에 사용됩니다
- 대화 이력은 세션 종료 후 삭제됩니다
- 시스템 설정에 따라 로그가 기록될 수 있습니다
