=========
범위 지정 검색
=========

숫자 등 범위 지정이 가능한 데이터를 필드에 저장한 경우 해당 필드에 대해 범위 지정 검색이 가능합니다.

사용 방법
------

범위 지정 검색을 하려면 "필드명:[값 TO 값]"을 검색 양식에 입력합니다.

예를 들어, content_length 필드에 대해 1k바이트에서 10k바이트 사이에 있는 문서를 검색하는 경우 다음과 같이 검색 양식에 입력합니다.

::

    content_length:[1000 TO 10000]

시간의 범위 지정 검색을 하려면 "last_modified:[일시1 TO 일시2]"(일시1＜일시2)를 검색 양식에 입력합니다.

일시는 ISO 8601을 기준으로 합니다.

.. list-table::

   * - 연월일 및 시분초 및 소수 부분
     - 현재 일시를 기준으로 하는 경우
   * - YYYY-MM-DDThh:mm:sssZ (예: 2012-12-02T10:45:235Z)
     - now(현재 일시), y(년), M(월), w(주), d(일), h(시), m(분), s(초)

now나 시각을 기준으로 한 경우 +, -(가산, 감산)이나 /(반올림) 같은 기호를 붙일 수 있습니다. 다만, 시각을 기준으로 한 경우 기호 사이에 ||를 넣어야 합니다.

/는 / 뒤의 단위로 반올림하는 기호입니다. now-1d/d는 오늘 몇 시에 실행하더라도 오늘 00:00에서 -1일 한 전날 00:00을 나타냅니다.

예를 들어, last_modified 필드에 대해 2016년 2월 21일 20시(현재 일시로 가정)부터 30일 전까지 업데이트된 문서를 검색하는 경우 다음과 같이 검색 양식에 입력합니다.

::

    last_modified:[now-30d TO now](=[2016-01-23T00:00:000Z+TO+2016-02-21T20:00:000Z(현재 일시)])
