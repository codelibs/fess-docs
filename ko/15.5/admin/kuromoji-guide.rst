===========
Kuromoji 사전
===========

개요
====

인명, 고유명사, 전문용어 등을 형태소 분석용으로 등록할 수 있습니다.

관리 방법
======

표시 방법
------

아래 그림의 Kuromoji 설정 목록 페이지를 열려면 왼쪽 메뉴의 [시스템 > 사전]을 선택한 후 kuromoji를 클릭합니다.

|image0|

편집하려면 설정 이름을 클릭합니다.

설정 방법
------

Kuromoji 설정 페이지를 열려면 신규 생성 버튼을 클릭합니다.

|image1|

설정 항목
------

토큰
::::::

형태소 분석에서 처리할 단어를 입력합니다.

분할
::::

단어가 복합어로 구성된 경우, 분할한 단어로 검색해도 히트되도록 할 수 있습니다.
예를 들어 "전문검색엔진"을 "전문 검색 엔진"으로 입력하면 분할한 단어로도 검색할 수 있습니다.

읽기
::::

토큰으로 입력한 단어의 읽기를 가타카나로 입력합니다.
분할을 수행한 경우 분할하여 입력합니다.
예를 들어 "ゼンブン ケンサク エンジン"과 같이 입력합니다.

품사
::::

입력한 단어의 품사를 입력합니다.

다운로드
=========

Kuromoji 사전 형식으로 다운로드할 수 있습니다.

업로드
=========

Kuromoji 사전 형식으로 업로드할 수 있습니다.
Kuromoji 사전 형식은 쉼표(,) 구분으로 "토큰,분할토큰,분할토큰의 읽기,품사"입니다.
분할 토큰은 공백으로 구분하여 분할합니다.
분할할 필요가 없는 경우 토큰과 분할 토큰은 같습니다.
예를 들어 다음과 같습니다.

::

    朝青龍,朝青龍,アサショウリュウ,カスタム名詞
    関西国際空港,関西 国際 空港,カンサイ コクサイ クウコウ,カスタム名詞
