======
유사 검색
======

유사 검색(퍼지 검색)
=================

검색어와 완전히 일치하지 않는 단어도 검색하는 경우 유사 검색을 사용할 수 있습니다. 철자 오류나 표기 차이 등으로 완전히 일치하지 않는 문서도 찾고 싶은 경우에 유용합니다. |Fess| 는 편집 거리(레벤슈타인 거리)에 기반한 유사 검색(퍼지 검색)을 지원합니다.

사용 방법
------

유사 검색을 적용하려는 검색어 뒤에 "~"를 추가합니다.

예를 들어, "Fess"라는 단어를 유사 검색하려는 경우 다음과 같이 검색 양식에 입력하면 "Fess"와 유사한 단어("Fes" 등)를 포함하는 문서를 검색할 수 있습니다.

::

    Fess~

유사함의 정도는 ``~`` 뒤에 0~2 사이의 숫자를 붙여서 지정할 수 있습니다. 숫자는 허용할 편집 거리(문자의 삽입・삭제・치환 횟수의 상한)를 나타내며, 생략한 경우에는 기본 편집 거리가 적용됩니다.

::

    Fess~1
    Fess~2

``~2`` 를 지정하면 ``Fess`` 와 최대 2문자까지의 차이를 허용하여 검색합니다.
