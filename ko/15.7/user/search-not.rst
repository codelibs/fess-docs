=======
NOT 검색
=======

특정 단어를 포함하지 않는 문서를 검색하는 경우 NOT 검색을 사용할 수 있습니다. 이는 제외 검색이라고도 하며, 특정 키워드를 검색 결과에서 제외하여 불필요한 결과(노이즈)를 줄이고 싶을 때 유용합니다.

사용 방법
------

NOT 검색은 제외하려는 단어 앞에 NOT을 붙여서 검색합니다. NOT은
반각 대문자로 입력해야 하며, 앞뒤로 공백이 필요합니다.

예를 들어, "검색어1"은 포함하지만 "검색어2"는 포함하지 않는 문서를 검색하려는 경우 다음과 같이 입력하여 검색합니다.

::

    검색어1 NOT 검색어2

NOT 대신, 제외하려는 단어 바로 앞에 ``-`` (반각 하이픈)를 붙이는 방법도 있습니다. 아래는 위와 같은 의미가 됩니다.

::

    검색어1 -검색어2
