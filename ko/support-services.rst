상용 지원
============

상용 지원에 대하여
====================

Fess는 Apache 라이센스로 제공되는 오픈소스로 개인이나 상용에서도 무료로 자유롭게 이용하실 수 있습니다.

`N2SM, Inc. <https://www.n2sm.net/>`__ 는 다음을 제공하고 있습니다.

- `N2 Search <https://www.n2sm.net/n2search.html>`__: Fess의 상용 제품 버전

- `검색 솔루션 지원 <https://www.n2sm.net/products/n2search/support.html>`__: Fess 및 Elasticsearch의 커스터마이즈 및 구축 지원 등

N2 Search
=========

`N2 Search <https://www.n2sm.net/n2search.html>`__ 는 Fess와 비교하여 다음 점이 다릅니다.

- 검색 성능 향상

- 디스크 사용량 최적화

- 관리 기능 보안 강화

- 신조어 대응 형태소 분석

지원 서비스
================

다음과 같은 서비스가 필요한 경우 `상용 지원(검색 솔루션 지원) <https://www.n2sm.net/products/n2search/support.html>`__ 을 검토하시기 바랍니다.

- 도입·운용 컨설팅

- 개별 요구사항에 의한 커스터마이즈 개발 (Fess 프로젝트에서는 범용적인 요구사항에는 대응하지만 개별적이라고 판단되는 요구사항은 대응하지 않습니다)

- 대량 문서 크롤링 대응 (50만 문서를 초과하는 경우 Fess/Elasticsearch에 대한 충분한 지식이 필요합니다)

- 클러스터링 구성

- 자사 기업 내 검색·사내 검색에 검색 결과 최적화

- Fess/Elasticsearch 서버 튜닝

- 인덱스 대상이 되지 않는 파일 조사·대응 (Fess 프로젝트에서는 의존하는 라이브러리가 읽을 수 없는 경우 그 이상의 지원은 하지 않으므로 필요한 경우 상용 지원으로 문의하시기 바랍니다)

- 고속 전문 검색 시스템 환경 구축

- 특정 버전 유지보수 (Fess 프로젝트에서는 최신 버전에 대한 수정만 수행합니다)

- 교육

- 기타

AWS Marketplace
===============

문서 수 상한 등의 기능 제한이 있지만 AWS Marketplace에서 N2 Search를 이용하실 수 있습니다.
중소기업용 문서 검색 엔진으로도 간단히 도입 가능합니다.
`N2 Search <https://aws.amazon.com/marketplace/pp/B014JFU5EW>`__ 를 참조하세요.
