================
검색 대상 파일
================

지원하는 파일 형식
============================

Fess가 크롤링 및 검색에 대해 확인한 파일 형식 목록입니다.

- 텍스트 (txt)

- XML (xml, xhtml, mm 등)

- HTML (html, htm)

- MS Office (doc, xls, ppt, docx, xlsx, pptx 등)

- PDF (pdf 등)

- 소스 코드 (js, c, h, java 등)

- 압축 파일 (gz, tar, zip 등)

- 리치 텍스트 (rtf)

- ePub

- 음성/이미지/동영상 (메타데이터 추출)

- mbox

- ai 파일 (PDF 호환)

Fess는 알려지지 않은 다양한 종류의 파일에서도 문자열 추출을 수행합니다. 위에 기재되지 않은 파일도 크롤링 및 검색할 수 있습니다. 확인하고 싶은 파일이 있으면 \ `검색 시스템용 테스트 데이터 리포지토리 <https://github.com/codelibs/fess-testdata>`__\ 에 풀 리퀘스트를 해주시기 바랍니다.

기타
======

다음 파일은 상용 지원으로 대응합니다.

-  일본 워드프로세서 (一太郎)

-  OASYS for Windows

-  DocuWorks

-  AutoCAD
