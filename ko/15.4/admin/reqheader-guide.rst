==============
요청 헤더
==============

개요
====

여기서는 요청 헤더에 관한 설정에 대해 설명합니다.
요청 헤더 기능은 문서를 크롤링하여 가져올 때의 요청에 추가하는 요청 헤더 정보입니다.
예를 들어, 인증 시스템에서 헤더 정보를 보고 특정 값이 있으면 자동으로 로그인한 상태로 만드는 경우 등에 사용할 수 있습니다.

관리 방법
======

표시 방법
------

아래 그림의 요청 헤더 설정 목록 페이지를 열려면 왼쪽 메뉴의 [크롤러 > 요청 헤더]를 클릭합니다.

|image0|

편집하려면 설정 이름을 클릭합니다.

설정 생성
--------

요청 헤더 설정 페이지를 열려면 신규 생성 버튼을 클릭합니다.

|image1|

설정 항목
------

이름
::::

요청에 추가할 때의 요청 헤더 이름을 지정합니다.

값
::

요청에 추가할 때의 요청 헤더 값을 지정합니다.

웹 설정
::::::::

요청 헤더를 추가할 웹 크롤링 설정 이름을 선택합니다.
선택한 크롤링 설정에만 요청 헤더가 추가됩니다.

설정 삭제
--------

목록 페이지의 설정 이름을 클릭하고 삭제 버튼을 클릭하면 확인 화면이 표시됩니다.
삭제 버튼을 누르면 설정이 삭제됩니다.
