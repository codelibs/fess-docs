============
페이지 디자인
============

개요
====

여기에서는 검색 화면의 디자인에 관한 설정에 대해 설명합니다.

설정 방법
======

표시 방법
------

아래 그림의 페이지 디자인 설정을 수행하기 위한 목록 페이지를 열려면 왼쪽 메뉴의 [시스템 > 페이지 디자인]을 클릭합니다.

|image0|


파일 관리자
---------------

검색 화면에서 사용 가능한 파일을 다운로드하거나 삭제할 수 있습니다.

페이지 파일 표시
---------------

검색 화면의 JSP 파일을 편집할 수 있습니다.
대상 JSP 파일의 편집 버튼을 눌러 해당 JSP 파일을 편집할 수 있습니다.
또한 기본값 사용 버튼을 누르면 설치 시의 JSP 파일로 편집할 수 있습니다.
편집 화면에서 업데이트 버튼으로 저장하면 변경 사항이 반영됩니다.

다음은 편집 가능한 페이지에 대한 설명입니다.

.. tabularcolumns:: |p{4cm}|p{8cm}|
.. list-table::

   * - /index.jsp
     - 검색 최상위 페이지의 JSP 파일입니다. 이 JSP 파일이 각 부분의 JSP 파일을 include 합니다.
   * - /header.jsp
     - 헤더의 JSP 파일입니다.
   * - /footer.jsp
     - 푸터의 JSP 파일입니다.
   * - /search.jsp
     - 검색 결과 목록 페이지의 JSP 파일입니다. 이 JSP 파일이 각 부분의 JSP 파일을 include 합니다.
   * - /searchResults.jsp
     - 검색 결과 목록 페이지의 검색 결과 부분을 표현하는 JSP 파일입니다. 검색 결과가 있을 때 사용되는 JSP 파일입니다. 검색 결과의 표현을 사용자 정의하려는 경우 변경합니다.
   * - /searchNoResult.jsp
     - 검색 결과 목록 페이지의 검색 결과 부분을 표현하는 JSP 파일입니다. 검색 결과가 없을 때 사용되는 JSP 파일입니다.
   * - /searchOptions.jsp
     - 검색 옵션 화면의 JSP 파일입니다.
   * - /advance.jsp
     - 상세 검색 화면의 JSP 파일입니다.
   * - /help.jsp
     - 도움말 페이지의 JSP 파일입니다.
   * - /error/error.jsp
     - 검색 오류 페이지의 JSP 파일입니다. 검색 오류의 표현을 사용자 정의하려는 경우 변경합니다.
   * - /error/notFound.jsp
     - 페이지를 찾을 수 없는 경우 표시되는 오류 페이지의 JSP 파일입니다.
   * - /error/system.jsp
     - 시스템 오류 시 표시되는 오류 페이지의 JSP 파일입니다.
   * - /error/redirect.jsp
     - HTTP 리다이렉트 발생 시 표시되는 오류 페이지의 JSP 파일입니다.
   * - /error/badRequest.jsp
     - 잘못된 요청 발생 시 표시되는 오류 페이지의 JSP 파일입니다.
   * - /cache.hbs
     - 검색 결과의 캐시를 표시하는 파일입니다.
   * - /login/index.jsp
     - 로그인 화면의 JSP 파일입니다.
   * - /profile/index.jsp
     - 사용자용 비밀번호 변경 화면의 JSP 파일입니다.
   * - /profile/newpassword.jsp
     - 관리자용 비밀번호 업데이트 화면의 JSP 파일입니다. 로그인 시 사용자명과 비밀번호가 같은 문자열인 경우 비밀번호 변경을 요청합니다.

업로드할 파일
------------------

검색 화면에서 사용할 파일을 업로드할 수 있습니다.
지원하는 이미지 파일명은 jpg, gif, png, css, js입니다.

파일 업로드
:::::::::::::::::

업로드할 파일을 지정합니다.

파일명(옵션)
:::::::::::::::::

업로드할 파일에 파일명을 지정하고자 하는 경우 사용합니다.
생략한 경우 업로드한 파일명이 사용됩니다.
예를 들어 logo.png를 지정하면 검색 최상위 페이지의 이미지가 변경됩니다.
