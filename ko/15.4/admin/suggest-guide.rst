============
추천 단어
============

개요
====

추천 단어 페이지는 키워드 추천에서 표시할 단어를 관리합니다.

관리 방법
======

표시 방법
------

아래 그림의 추천 단어 목록 페이지를 열려면 왼쪽 메뉴의 [추천 > 추천 단어]를 클릭합니다.

|image0|

삭제하려면 삭제 버튼을 클릭합니다.


단어 종류
::::::

- 전체: 등록된 모든 단어
- 문서: 인덱싱된 문서에서 생성된 단어
- 검색어: 검색 로그에서 생성된 단어

단어 수
:::::

등록된 단어 수
