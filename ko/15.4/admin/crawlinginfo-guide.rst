=========
크롤링 정보
=========

개요
====

크롤링의 실행 결과가 기록되며, 크롤링 정보는 이 관리 화면에서 확인할 수 있습니다.

관리 방법
======

목록
====

크롤링이 실행될 때마다 크롤링 정보로 기록됩니다.
목록에서는 실행한 크롤링의 세션 이름과 실행한 시간을 확인할 수 있습니다.
크롤링 정보의 세부 정보를 확인하려면 대상 크롤링 정보를 클릭합니다.

|image0|

세부 정보
====

목록의 크롤링 정보 세션 ID를 클릭하면 대상 크롤링 정보 세부 정보가 표시됩니다.

|image1|

항목 목록
------

세션 ID
::::::::::

크롤링 실행 시의 세션 ID.

크롤러 시작 시간
:::::::::::::::

크롤링 전체가 시작한 시각.

크롤링 시작 시간 (웹/파일)
::::::::::::::::::::::::::::

웹 및 파일 시스템 크롤링이 시작된 시각.

크롤링 시작 시간 (데이터 저장소)
::::::::::::::::::::::::::

데이터 저장소 크롤링이 시작한 시각.

크롤링 종료 시간 (웹/파일)
::::::::::::::::::::::::::::

웹 및 파일 시스템 크롤링이 종료한 시각.

크롤링 실행 시간 (데이터 저장소)
::::::::::::::::::::::::::

데이터 저장소 크롤링의 실행 시간(밀리초).

::::::::::::::::::::::::::::::

인덱스 크기 (데이터 저장소)
:::::::::::::::::::::::::::

인덱싱된 문서의 수.

크롤링 종료 시간 (데이터 저장소)
::::::::::::::::::::::::::

데이터 저장소 크롤링이 종료한 시각.

크롤러 상태

크롤러 종료 시간
:::::::::::::::

크롤링 전체가 종료한 시각.

크롤러 실행 시간
:::::::::::::::

크롤링 전체의 실행 시간(밀리초).
