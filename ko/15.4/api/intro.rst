API 사용 전제 조건
============

|Fess| 의 API를 사용하기 전에 다음 설정이 필요합니다.

- |Fess| 15.4 이상이 설치되어 정상적으로 실행 중일 것
- 관리 화면에서 API별로 필요한 설정을 활성화할 것(각 API 문서 참조)
- 네트워크를 통해 API 엔드포인트에 액세스 가능할 것
