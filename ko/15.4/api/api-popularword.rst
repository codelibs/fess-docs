==========
인기 검색어 API
==========

인기 검색어 목록 조회
============

요청
--

==================  ====================================================
HTTP 메서드         GET
엔드포인트        ``/api/v1/popular-words``
==================  ====================================================

|Fess| 에, ``http://<Server Name>/api/v1/popular-words?seed=123`` 요청을 전송하여, |Fess| 에 등록된 인기 검색어 목록을 JSON 형식으로 받을 수 있습니다.
인기 검색어 API를 사용하려면 관리 화면의 시스템 > 일반 설정에서 인기 검색어 응답을 활성화해야 합니다.

요청 매개변수
-------

사용 가능한 요청 매개변수는 다음과 같습니다.

.. tabularcolumns:: |p{3cm}|p{12cm}|
.. list-table:: 요청 매개변수

   * - seed
     - 인기 검색어를 조회하는 시드(이 값에 따라 다른 패턴의 단어를 조회할 수 있습니다)
   * - label
     - 필터링된 레이블명
   * - field
     - 인기 검색어를 생성하는 필드명


응답
--

다음과 같은 응답이 반환됩니다.

::

    {
      "record_count": 3,
      "data": [
        "python",
        "java",
        "javascript"
      ]
    }

각 요소는 다음과 같습니다.

.. tabularcolumns:: |p{3cm}|p{12cm}|
.. list-table:: 응답 정보

   * - record_count
     - 인기 검색어 등록 건수
   * - data
     - 인기 검색어 배열

오류 응답
=====

인기 검색어 API가 실패한 경우 다음과 같은 오류 응답이 반환됩니다.

.. tabularcolumns:: |p{4cm}|p{11cm}|
.. list-table:: 오류 응답

   * - 상태 코드
     - 설명
   * - 400 Bad Request
     - 요청 매개변수가 올바르지 않은 경우
   * - 500 Internal Server Error
     - 서버 내부 오류가 발생한 경우
