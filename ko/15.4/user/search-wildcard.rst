==============
와일드카드 검색
==============

검색어 내에서 1개 문자 또는 여러 문자의 와일드카드를 사용할 수 있습니다. ?는 1개 문자의 와일드카드로 지정할 수 있으며, \*는 여러 문자의 와일드카드로 지정할 수 있습니다. 와일드카드를 사용할 수 있는 대상은 단어입니다. 문장에 대한 와일드카드 검색은 할 수 없습니다.

사용 방법
------

1개 문자의 와일드카드를 사용하는 경우 다음과 같이 ?를 사용합니다.

::

    te?t

위의 경우 text나 test 등 1개 문자의 와일드카드로 처리됩니다.

여러 문자의 와일드카드를 사용하는 경우 다음과 같이 \*를 사용합니다.

::

    test*

위의 경우 test, tests나 tester 등 여러 문자의 와일드카드로 처리됩니다. 또한,

::

    te*t

와 같이 검색어 내에 사용할 수도 있습니다.

사용 조건
------

와일드카드는 인덱스에 등록된 문자열을 대상으로 사용됩니다.
따라서 bi-gram 등으로 인덱스를 생성한 경우 일본어는 의미 없는 고정 문자열 길이로 처리되므로 일본어 와일드카드는 예상대로 작동하지 않습니다.
일본어로 와일드카드를 사용하는 경우 형태소 분석을 사용하는 필드에서 사용하십시오.
