========
역할 검색
========

|Fess| 는 사용자 관리 기능을 제공하며, 로그인한 사용자는 자신이 속한 역할을 기준으로 검색할 수 있습니다. |Fess| 에서 관리하는 사용자는 로그인 후 역할 검색을 이용하거나 자신의 비밀번호를 변경할 수 있습니다.

역할 검색에서는 콘텐츠에 설정된 권한(역할·그룹·사용자)과 검색하는 사용자가 가진 권한을 대조하여, 접근이 허용된 콘텐츠만 검색 결과에 표시합니다. 역할과 사용자를 만드는 방법, 콘텐츠에 권한을 할당하는 방법 등 역할 기반 검색 설정에 대해서는 :doc:`../config/security-role` 를 참조하십시오.


검색 방법
------

역할을 설정하고 크롤링 및 인덱싱한 상태라면 검색 결과는 해당 역할을 보유한 사용자에게만 표시할 수 있습니다.
사용자가 로그인한 경우에는 그 사용자가 속한 역할이나 그룹을 기준으로 검색이 수행됩니다.
로그인하지 않은 경우에는 guest 사용자로 검색되며, guest 대상으로 공개된 콘텐츠만 표시됩니다.

로그인
------

검색 화면 상단에 표시되는 로그인 링크를 클릭하면 로그인 화면이 표시됩니다. 사용자 이름과 비밀번호를 입력하여 로그인하면 검색 화면으로 돌아가고, 이후의 검색은 그 사용자가 속한 역할을 기준으로 수행됩니다.

.. note::
    싱글 사인온이나 LDAP 와 연동되어 있는 경우에는 각각의 인증 방식으로 로그인합니다. 또한 로그인 링크의 표시 여부는 설정으로 변경할 수 있습니다.

비밀번호 변경
------------

로그인 후 검색 화면 상단에 표시되는 사용자 이름을 클릭하면 메뉴가 표시됩니다.

|image0|

메뉴에서 "비밀번호 변경"을 클릭하면 비밀번호 변경 화면이 표시됩니다.

|image1|

현재 비밀번호, 새 비밀번호, 확인용 새 비밀번호(재입력)를 입력하고 갱신 버튼을 클릭하면 비밀번호가 갱신됩니다.
비밀번호 변경 후 뒤로 버튼을 클릭하면 검색 화면으로 돌아갈 수 있습니다.

.. note::
    "비밀번호 변경" 메뉴는 |Fess| 에서 관리하는 사용자(및 편집이 허용된 LDAP 사용자)에게만 표시됩니다. 싱글 사인온으로 인증한 사용자에게는 표시되지 않습니다.
    새 비밀번호에는 길이나 사용할 수 있는 문자 종류 등의 비밀번호 정책이 적용될 수 있습니다.

로그아웃
--------

로그인한 상태에서 검색 화면 상단에 표시되는 사용자 이름을 클릭하고 메뉴에서 "로그아웃"을 선택하면 로그아웃할 수 있습니다.
관리자 권한을 가진 사용자의 경우, 같은 메뉴에서 "관리"를 선택하여 관리 화면으로 이동할 수도 있습니다.
