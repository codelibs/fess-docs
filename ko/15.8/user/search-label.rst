===========
라벨 지정 검색
===========

라벨 지정 검색 (카테고리 검색)
=======================

검색 대상 문서에 카테고리를 분류하기 위한 라벨 정보를 추가함으로써 검색 시 라벨을 지정한 필터링 검색이 가능합니다. 라벨을 이용하면 예를 들어 부서나 사이트, 문서 종류별로 검색 범위를 제한할 수 있습니다.

라벨은 관리 화면에서 미리 등록해 두면 검색 화면에서 라벨을 이용한 필터링 검색이 가능해집니다. 이용 가능한 라벨은 검색 시 풀다운 메뉴에서 여러 개를 선택할 수 있으며, 여러 라벨을 선택한 경우 그중 하나라도 부여된 문서가 검색 대상이 됩니다. 라벨을 등록하지 않은 경우 라벨 풀다운 박스는 표시되지 않습니다.

.. note::
    라벨에는 권한(퍼미션)을 설정할 수 있으므로, 검색하는 사용자가 접근이 허용된 라벨만 풀다운에 표시됩니다. 또한 가상 호스트나 로케일(언어)에 따라 표시되는 라벨이 달라지는 경우도 있습니다. 따라서 라벨을 등록해 두었더라도 사용자에 따라 풀다운에 표시되지 않을 수 있습니다.

라벨은 라벨을 부여할 대상을 URL 경로의 정규 표현식으로 지정하여 정의합니다. 라벨 등록 방법과 설정 항목에 대해서는 :doc:`라벨 관리 가이드 <../admin/labeltype-guide>` 를 참조하십시오.

사용 방법
------

검색 시 라벨 정보를 선택할 수 있습니다. 라벨 정보는 옵션 버튼을 클릭하면 표시되는 검색 옵션에서 선택할 수 있습니다.

|image0|

라벨을 설정하고 인덱스를 생성하면 라벨이 설정된 문서별로 검색할 수 있습니다. 라벨을 지정하지 않은 검색은 일반적인 전체 검색과 동일합니다.

문서에 대한 라벨 부여는 크롤링하여 인덱스를 생성할 때 문서의 URL과 라벨에 설정된 경로를 대조하여 이루어집니다. 따라서 라벨 정의(대상으로 하는 경로나 제외할 경로)를 추가·변경한 경우, 그 내용은 이미 인덱싱된 문서에는 자동으로 반영되지 않습니다. 변경 사항을 반영하려면 대상 문서를 다시 크롤링하거나, 스케줄러에 등록된 "Label Updater" 작업을 실행하여 인덱스를 업데이트하십시오.
