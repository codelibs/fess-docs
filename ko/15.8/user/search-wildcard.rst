==============
와일드카드 검색
==============

검색어 내에서 1개 문자 또는 여러 문자의 와일드카드를 사용할 수 있습니다. ?는 1개 문자의 와일드카드로 지정할 수 있으며, \*는 여러 문자의 와일드카드로 지정할 수 있습니다. 와일드카드를 사용할 수 있는 대상은 단어입니다. 문장에 대한 와일드카드 검색은 할 수 없습니다.

사용 방법
------

1개 문자의 와일드카드를 사용하는 경우 다음과 같이 ?를 사용합니다.

::

    te?t

위의 경우 text나 test 등, ? 위치의 임의의 한 문자에 일치합니다.

여러 문자의 와일드카드를 사용하는 경우 다음과 같이 \*를 사용합니다.

::

    test*

위의 경우 test, tests나 tester 등, \* 위치의 0자 이상의 임의의 문자열에 일치합니다. 또한,

::

    te*t

와 같이 검색어 중간에서도 사용할 수 있습니다. 더 나아가,

::

    *test

와 같이 검색어 앞부분에서도 와일드카드를 사용할 수 있습니다.

필드를 지정하여 와일드카드 검색을 수행할 수도 있습니다. 다음 예에서는 title 필드에서 te로 시작하고 t로 끝나는 단어를 포함하는 문서를 검색합니다.

::

    title:te*t

필드를 지정하지 않은 경우에는 title 및 content 필드를 대상으로 검색합니다.

사용 조건
------

와일드카드 검색을 사용할 때는 다음 사항에 주의하십시오.

* 와일드카드는 인덱스에 등록된 문자열(토큰)을 대상으로 일치합니다. 검색어는 다시 분석되지 않으므로, bi-gram 등으로 인덱스를 생성한 경우 일본어는 의미를 갖지 않는 고정 길이 문자열로 처리되어 일본어에서의 와일드카드는 예상한 대로 동작하지 않습니다. 일본어로 와일드카드를 사용하는 경우에는 형태소 분석을 사용하는 필드에서 사용하십시오.
* 와일드카드 패턴은 대문자와 소문자를 구분합니다. 인덱스에 등록되는 단어는 일반적으로 소문자로 변환되므로 패턴에는 소문자를 사용하십시오. 예를 들어 ``Test*`` 는 소문자로 변환되어 등록된 ``test`` 와 일치하지 않습니다.
* 검색어 앞부분에 와일드카드를 붙인 검색(예: ``*test``)은 인덱스 내의 모든 단어를 탐색하므로 처리에 시간이 걸릴 수 있습니다.
