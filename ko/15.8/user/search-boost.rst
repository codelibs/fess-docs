=========
부스트 검색
=========

부스트 검색(가중치 검색)
====================

검색어 중 특정 검색어를 우선하고 싶은 경우에는 부스트 검색을 사용합니다. 부스트 검색을 사용하면 검색어의 중요도에 따른 검색이 가능해집니다.

사용 방법
----------

부스트 검색을 사용하려면 검색어 바로 뒤에 "^부스트값" 형식으로 부스트 값(가중치 값)을 지정합니다. 검색어와 부스트 값 사이에는 공백을 넣지 않습니다.

예를 들어 "사과 귤"을 검색할 때 "사과"를 더 우선하고 싶은 경우에는 다음과 같이 검색 양식에 입력합니다.

::

    사과^100 귤

이 예에서는 "사과"에만 부스트 값이 적용되고, "귤"에는 적용되지 않습니다.

부스트 값
----------

부스트 값에는 숫자를 지정합니다. 정수뿐만 아니라 ``2.5``\ 와 같은 소수도 지정할 수 있습니다.

- 1보다 큰 값을 지정하면 해당 검색어의 가중치가 증가합니다.
- 0보다 크고 1 미만인 값(예: ``0.5``)을 지정하면 해당 검색어의 가중치가 감소합니다.

부스트 값은 검색 결과의 점수를 절대적으로 결정하는 것이 아니라, 다른 검색어와의 상대적인 가중치를 조정하는 것입니다.

다른 검색 구문과의 조합
--------------------

부스트 검색은 다른 검색 구문과 조합하여 사용할 수 있습니다. 예를 들어 필드를 지정한 검색어에 부스트 값을 추가할 수도 있습니다.

::

    title:사과^100 귤

이 예에서는 title 필드에 "사과"가 포함된 문서가 우선됩니다.
