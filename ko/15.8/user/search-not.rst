=======
NOT 검색
=======

특정 단어를 포함하지 않는 문서를 검색하는 경우 NOT 검색을 사용할 수 있습니다.

사용 방법
------

NOT 검색은 제외하려는 단어 앞에 NOT을 붙여서 검색합니다. NOT은
반각 대문자로 입력해야 하며, 앞뒤로 공백이 필요합니다.

예를 들어, "검색어1"은 포함하지만 "검색어2"는 포함하지 않는 문서를 검색하려는 경우 다음과 같이 입력하여 검색합니다.

::

    검색어1 NOT 검색어2

.. note::

    NOT은 반각 대문자로 입력해야 합니다. ``not``\ 처럼 소문자로 입력한 경우에는 연산자로 취급되지 않고, "not"이라는 일반 검색어로 검색됩니다. 또한, 제외하고 싶은 단어 바로 앞에 ``-``\ 를 붙여 ``검색어1 -검색어2``\ 처럼 작성해도 NOT과 같은 의미가 됩니다.

괄호 ``( )``\ 를 사용하면 NOT 검색과 다른 검색 조건을 조합할 수 있습니다. 예를 들어 "검색어1" 또는 "검색어2" 중 하나를 포함하고, "검색어3"은 포함하지 않는 문서를 검색하려는 경우 다음과 같이 입력합니다.

::

    (검색어1 OR 검색어2) NOT 검색어3

필드를 지정하여 NOT 검색을 수행할 수도 있습니다. 다음 예에서는 title 필드에 "검색어1"을 포함하고, title 필드에 "검색어2"는 포함하지 않는 문서를 검색합니다.

::

    title:검색어1 NOT title:검색어2
