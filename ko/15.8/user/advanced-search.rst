==========
상세 검색
==========

상세 검색 화면에서는 여러 조건을 조합하여 더욱 복잡한 검색을 수행할 수 있습니다.

사용 방법
----------

상세 검색 화면은 검색 화면의 검색 옵션에 있는 「상세 검색」 링크에서 열 수 있습니다. 검색 결과 화면에서 열면 이미 입력한 검색 키워드가 「모든 단어 포함」 필드로 인계됩니다.

|image0|

각 필드에 조건을 입력하고 페이지 하단의 검색 버튼을 클릭합니다. 입력한 여러 필드는 하나의 검색 조건으로 조합됩니다.

항목 목록
----------

모든 단어 포함
::::::::::::::::

입력한 모든 단어를 포함하는 문서를 검색합니다(AND 검색). 여러 단어를 공백으로 구분하여 지정할 수 있습니다.

어순 포함 완전 일치
::::::::::::::::

입력한 문자열을 어순을 포함하여 그대로 포함하는 문서를 검색합니다(구문 검색). 공백을 포함한 문자열 전체를 하나의 구문으로 취급하므로 개별 단어로 나누어 검색할 수 없습니다.

단어 중 하나라도 포함
::::::::::::::::

입력한 단어 중 하나라도 포함하는 문서를 검색합니다(OR 검색). 여러 단어를 공백으로 구분하여 지정할 수 있습니다.

제외할 단어
::::::::::::::::

입력한 단어를 포함하지 않는 문서를 검색합니다. 여러 단어를 공백으로 구분하여 지정할 수 있으며, 지정한 모든 단어가 검색 결과에서 제외됩니다.

표시 건수
::::::::::::::::

한 페이지에 표시되는 검색 결과 건수를 지정합니다. 10, 20, 30, 40, 50, 100 중에서 선택할 수 있습니다. 지정하지 않으면 기본 표시 건수(초깃값 10)가 사용됩니다.

정렬
::::::::::::::::

검색 결과를 정렬하는 기준을 지정합니다. 점수 순, 파일 이름, 날짜, 크기, 최종 갱신 일시 중에서 각각 오름차순 또는 내림차순으로 선택할 수 있습니다. 해당 기능이 활성화된 경우에는 클릭 수 또는 즐겨찾기 수로 정렬할 수도 있습니다.

우선 언어
::::::::::::::::

검색 결과에서 우선할 언어를 지정합니다. 여러 언어를 선택할 수 있습니다. 「모든 언어」를 선택하면 특정 언어를 우선하지 않습니다.

라벨
::::::::::::::::

라벨로 검색을 필터링합니다. 여러 라벨을 선택할 수 있습니다. 라벨이 등록되어 있지 않거나 현재 사용자가 볼 수 있는 라벨이 없는 경우에는 표시되지 않습니다.

업데이트 일시
::::::::::::::::

문서의 업데이트 일시로 검색을 필터링합니다. 「지정 없음」, 「24시간 이내」, 「1주일 이내」, 「1개월 이내」, 「1년 이내」 중에서 선택할 수 있습니다.

파일 형식
::::::::::::::::

문서의 파일 형식으로 검색을 필터링합니다. 「모든 형식」, 「HTML」, 「PDF」, 「MS Word」, 「MS Excel」, 「MS PowerPoint」 중에서 선택할 수 있습니다.

검색 대상
::::::::::::::::

검색할 대상을 지정합니다. 「페이지 전체」, 「페이지 내 제목」, 「페이지 내 URL」 중에서 선택할 수 있습니다. 제목 또는 URL 옵션을 선택하면 입력한 단어를 각각 제목 또는 URL에서만 검색합니다.

사이트 또는 도메인
::::::::::::::::

입력한 사이트 또는 도메인으로 검색을 필터링합니다. 특정 사이트나 도메인에 포함된 문서만 검색하고 싶을 때 지정합니다.
