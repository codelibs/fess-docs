Recherche par étiquette
===========================

Recherche par étiquette (recherche par catégorie)
==================================================

En ajoutant des informations d'étiquette pour catégoriser les documents cibles de la recherche, il est possible d'effectuer une recherche filtrée en spécifiant une étiquette lors de la recherche.

Les informations d'étiquette peuvent être enregistrées dans l'écran d'administration, permettant ainsi la recherche par étiquettes sur l'écran de recherche. Les étiquettes disponibles peuvent être sélectionnées dans un menu déroulant lors de la recherche. Si aucune étiquette n'est enregistrée, le menu déroulant des étiquettes ne sera pas affiché.

Utilisation
-----------

Vous pouvez sélectionner les informations d'étiquette lors de la recherche. Les informations d'étiquette peuvent être sélectionnées dans la boîte de dialogue des options de recherche affichée en cliquant sur le bouton Options.

|image0|

En créant un index avec des étiquettes configurées, vous pouvez effectuer une recherche pour chaque document auquel une étiquette est attribuée. Une recherche sans spécifier d'étiquette est une recherche complète normale. Si les informations d'étiquette sont modifiées, il est nécessaire de mettre à jour l'index.
