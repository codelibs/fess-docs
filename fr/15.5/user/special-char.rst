====================
Caractères spéciaux
====================

Les caractères suivants sont traités comme des caractères spéciaux dans les termes de recherche. Pour utiliser ces caractères comme caractères de recherche, ils doivent être échappés.

::

    + - && || ! ( ) { } [ ] ^ " ~ * ? : \ /


Utilisation
-----------

Échappez avec \ ou entourez le terme de recherche avec ".

::

    aaa\/bbb
    "aaa/bbb"
