================
Fichiers recherchables
================

Formats de fichiers pris en charge
============================

Voici la liste des formats de fichiers que Fess a confirmés pour le crawling et la recherche.

- Texte (txt)

- XML (xml, xhtml, mm, etc.)

- HTML (html, htm)

- MS Office (doc, xls, ppt, docx, xlsx, pptx, etc.)

- PDF (pdf, etc.)

- Code source (js, c, h, java, etc.)

- Fichiers compressés (gz, tar, zip, etc.)

- Rich Text (rtf)

- ePub

- Audio/Image/Vidéo (extraction de métadonnées)

- mbox

- Fichiers ai (compatible PDF)

Fess extrait également du texte à partir de divers types de fichiers inconnus. Les fichiers non répertoriés ci-dessus peuvent également être crawlés et recherchés. Si vous souhaitez vérifier un fichier spécifique, veuillez soumettre une pull request au \ `référentiel de données de test pour systèmes de recherche <https://github.com/codelibs/fess-testdata>`__\ .
