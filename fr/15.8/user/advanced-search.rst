==================
Recherche avancée
==================

L'écran de recherche avancée permet d'effectuer des recherches plus complexes en combinant plusieurs conditions.

Utilisation
-----------

L'écran de recherche avancée est accessible depuis le lien « Recherche avancée » dans les options de recherche de l'écran de recherche. Lorsqu'il est ouvert depuis l'écran des résultats de recherche, les mots-clés déjà saisis sont reportés dans le champ « Tous les mots ».

|image0|

Saisissez les conditions dans chaque champ, puis cliquez sur le bouton Rechercher en bas de la page. Les différents champs saisis sont combinés en une seule condition de recherche.

Liste des champs
----------------

Tous les mots
:::::::::::::

Recherche les documents contenant tous les mots saisis (recherche ET). Vous pouvez spécifier plusieurs mots séparés par des espaces.

Correspondance exacte incluant l'ordre des mots
:::::::::::::::::::::::::::::::::::::::::::::::::

Recherche les documents contenant exactement la chaîne de caractères saisie, ordre des mots compris (recherche par expression). Comme l'ensemble de la chaîne, espaces compris, est traité comme une seule expression, il n'est pas possible de la scinder en mots distincts pour la recherche.

N'importe quel mot
::::::::::::::::::

Recherche les documents contenant l'un quelconque des mots saisis (recherche OU). Vous pouvez spécifier plusieurs mots séparés par des espaces.

Mots à exclure
::::::::::::::

Recherche les documents ne contenant aucun des mots saisis. Vous pouvez spécifier plusieurs mots séparés par des espaces ; tous les mots spécifiés sont exclus des résultats de recherche.

Nombre de résultats
:::::::::::::::::::

Permet de spécifier le nombre de résultats de recherche affichés par page. Vous pouvez choisir parmi 10, 20, 30, 40, 50 ou 100. Si aucune valeur n'est spécifiée, le nombre d'affichage par défaut est utilisé (valeur initiale : 10).

Tri
:::

Permet de spécifier le critère de tri des résultats de recherche. Vous pouvez choisir parmi « par score », « par nom de fichier », « par date », « par taille » et « par dernière modification », chacun dans l'ordre croissant ou décroissant. Le tri « par nombre de clics » ou « par nombre de favoris » est disponible lorsque la fonctionnalité correspondante est activée.

Langue préférée
:::::::::::::::

Permet de spécifier la langue préférée des résultats de recherche. Vous pouvez sélectionner plusieurs langues. Si « Toutes les langues » est sélectionné, aucune langue particulière n'est privilégiée.

Étiquette
:::::::::

Permet de filtrer la recherche par étiquette. Vous pouvez sélectionner plusieurs étiquettes. Ce champ n'apparaît pas si aucune étiquette n'est enregistrée ou si aucune étiquette n'est consultable par l'utilisateur actuel.

Date de mise à jour
:::::::::::::::::::

Permet de filtrer la recherche par date et heure de mise à jour du document. Vous pouvez choisir parmi « N'importe quand », « Dernières 24 heures », « Semaine dernière », « Mois dernier » et « Année dernière ».

Format de fichier
:::::::::::::::::

Permet de filtrer la recherche par format de fichier du document. Vous pouvez choisir parmi « Tout format », « HTML », « PDF », « MS Word », « MS Excel » et « MS PowerPoint ».

Cible de recherche
::::::::::::::::::

Permet de spécifier l'emplacement de la recherche. Vous pouvez choisir parmi « N'importe où dans la page », « dans le titre de la page » et « dans l'URL de la page ». Si l'option titre ou URL est sélectionnée, les mots saisis sont recherchés uniquement dans le titre ou dans l'URL, respectivement.

Site ou domaine
:::::::::::::::

Permet de filtrer la recherche par le site ou le domaine saisi. Spécifiez-le lorsque vous souhaitez rechercher uniquement les documents contenus dans un site ou un domaine particulier.
