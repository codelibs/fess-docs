============
Introduction
============

À propos de ce document
=======================

Ce guide d'installation décrit les procédures d'installation, de démarrage, de mise à niveau et de désinstallation de |Fess|.
Veuillez sélectionner la méthode d'installation appropriée en fonction de votre environnement.

Public cible
============

Ce document s'adresse aux utilisateurs chargés de l'installation et de la désinstallation de |Fess|.

Avant de commencer
==================

Ce document présente les méthodes d'installation de |Fess|. Les connaissances suivantes sont requises :

- Connaissances de base de l'administration de serveurs
- Utilisation de la ligne de commande sous Linux ou Windows
- Configuration de base des réseaux et pare-feu
- Connaissances de base de Docker (si vous utilisez la version Docker)

Organisation du document
========================

Ce guide d'installation est organisé selon les sections suivantes :

- **Configuration requise** : Configuration matérielle et logicielle nécessaire pour exécuter |Fess|
- **Installation** : Procédures d'installation par plateforme

  - Installation pour Linux (TAR.GZ/RPM/DEB)
  - Installation pour Windows (ZIP)
  - Installation pour Docker

- **Démarrage et arrêt** : Démarrage, arrêt et configuration initiale du serveur |Fess|
- **Mise à niveau** : Procédure de mise à niveau d'une version antérieure vers la dernière version
- **Désinstallation** : Procédure de désinstallation complète de |Fess|
- **Configuration de la sécurité** : Configuration de sécurité recommandée pour les environnements de production
- **Dépannage** : Problèmes courants et solutions
