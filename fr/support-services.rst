==================
Support commercial
==================

À propos du support commercial
==============================

Fess est un logiciel open source fourni sous licence Apache et peut être utilisé librement et gratuitement à des fins personnelles ou commerciales.

`N2SM, Inc. <https://www.n2sm.net/>`__ propose ce qui suit :

- `N2 Search <https://www.n2sm.net/n2search.html>`__ : Version produit commercial de Fess

- `Support de solutions de recherche <https://www.n2sm.net/products/n2search/support.html>`__ : Personnalisation de Fess et OpenSearch, assistance à la construction, etc.

N2 Search
=========

`N2 Search <https://www.n2sm.net/n2search.html>`__ diffère de Fess sur les points suivants :

- Amélioration des performances de recherche

- Optimisation de l'utilisation du disque

- Renforcement de la sécurité des fonctions d'administration

- Analyse morphologique adaptée aux nouveaux mots

Services de support
===================

Si vous avez besoin de services tels que ceux énumérés ci-dessous, veuillez envisager le `support commercial (support de solutions de recherche) <https://www.n2sm.net/products/n2search/support.html>`__ :

- Conseil en déploiement et exploitation

- Développement de personnalisations selon des exigences individuelles (le projet Fess traite les exigences génériques, mais ne traite pas les exigences considérées comme individuelles)

- Support pour le crawl de grands volumes de documents (pour les projets dépassant 500 000 documents, une connaissance approfondie de Fess/OpenSearch est nécessaire)

- Configuration en cluster

- Optimisation des résultats de recherche pour votre recherche d'entreprise/recherche interne

- Optimisation des serveurs Fess/OpenSearch

- Enquête et traitement des fichiers non indexés (le projet Fess ne prend pas en charge les cas où les bibliothèques dépendantes ne peuvent pas lire les fichiers ; si nécessaire, veuillez consulter le support commercial)

- Construction d'un environnement de système de recherche plein texte haute performance

- Maintenance de versions spécifiques (le projet Fess n'effectue que des corrections sur la dernière version)

- Formation

- Autres

Foire aux questions
====================

Q : Quelle est la différence entre l'édition open source et l'édition commerciale ?
-------------------------------------------------------------------------------------

R : L'édition open source (Fess) est gratuite, mais le support technique repose sur la communauté.
L'édition commerciale (N2 Search) inclut des performances de recherche améliorées, des fonctionnalités de sécurité renforcées et un support professionnel.

Q : Puis-je essayer avant d'acheter ?
--------------------------------------

R : Oui. Le Fess open source peut être téléchargé et essayé immédiatement depuis la `page de téléchargement <downloads.html>`__.
Pour N2 Search, veuillez nous contacter via le formulaire de demande pour discuter d'un essai.

Q : Puis-je migrer de Fess vers N2 Search ?
---------------------------------------------

R : Oui, la migration est possible. Pour les procédures de migration détaillées, veuillez contacter le support commercial.
Nous vous guiderons sur la migration des données d'index et le transfert de configuration.

Q : Les contrats de support sont-ils annuels ?
-----------------------------------------------

R : Les durées des contrats de support sont flexibles et peuvent être adaptées à vos besoins.
Pour plus de détails, veuillez nous contacter via le `formulaire de contact N2SM <https://www.n2sm.net/contact.html>`__.

Q : Fess peut-il être utilisé dans des environnements cloud ?
--------------------------------------------------------------

R : Oui, il peut être utilisé sur les principales plateformes cloud telles qu'AWS, Azure et GCP.
Pour un déploiement basé sur Docker, consultez le `Guide d'installation Docker <15.5/install/install-docker.html>`__.
