Politique de versionnage
====================

Les versions sont fournies au format X.Y.Z.

-  X : Version majeure. La version augmente lorsque le schéma Elasticsearch de Fess est mis à jour.

-  Y : Version mineure. La version augmente en cas d'ajout de fonctionnalités ou de corrections de bugs sans mise à jour du schéma Elasticsearch de Fess.

-  Z :
   Version micro. La version augmente en cas de correction de bugs critiques.
   (Même si ce n'est pas critique, la version peut augmenter à la demande des utilisateurs. Traité au cas par cas)
