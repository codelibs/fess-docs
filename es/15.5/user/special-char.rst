=====================
Caracteres especiales
=====================

Los siguientes caracteres se tratan como caracteres especiales en los términos de búsqueda. Si desea utilizar estos caracteres como caracteres de búsqueda, debe escaparlos.

::

    + - && || ! ( ) { } [ ] ^ " ~ * ? : \ /


Cómo utilizar
-------------

Escape con \ o encierre el término de búsqueda entre comillas ".

::

    aaa\/bbb
    "aaa/bbb"
