============
Introducción
============

Acerca de este Documento
=========================

Esta guía de instalación describe los procedimientos de instalación, inicio, actualización y desinstalación de |Fess|.
Seleccione el método de instalación apropiado según su entorno.

Audiencia Objetivo
==================

Este documento está dirigido a usuarios responsables de la instalación y desinstalación de |Fess|.

Antes de Leer
=============

Este documento presenta el método de instalación de |Fess|. Se requieren los siguientes conocimientos:

- Conocimientos básicos de administración de servidores
- Operaciones de línea de comandos en Linux o Windows
- Configuración básica de redes y cortafuegos
- Conocimientos básicos de Docker (si utiliza la versión Docker)

Estructura del Documento
=========================

Esta guía de instalación está organizada en las siguientes secciones:

- **Requisitos del Sistema**: Requisitos de hardware y software necesarios para ejecutar |Fess|
- **Instalación**: Procedimientos de instalación por plataforma

  - Instalación para Linux (TAR.GZ/RPM/DEB)
  - Instalación para Windows (ZIP)
  - Instalación para Docker

- **Inicio y Detención**: Inicio, detención y configuración inicial del servidor |Fess|
- **Actualización**: Procedimientos de actualización desde versiones anteriores a la última versión
- **Desinstalación**: Procedimientos de desinstalación completa de |Fess|
- **Configuración de Seguridad**: Configuraciones de seguridad recomendadas para entornos de producción
- **Solución de Problemas**: Problemas comunes y soluciones
