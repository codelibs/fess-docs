Archivos Objetivo de Búsqueda
================

Formatos de Archivo Compatibles
============================

Esta es una lista de formatos de archivo que Fess ha verificado para rastreo y búsqueda.

- Texto (txt)

- XML (xml, xhtml, mm, etc.)

- HTML (html, htm)

- MS Office (doc, xls, ppt, docx, xlsx, pptx, etc.)

- PDF (pdf, etc.)

- Código fuente (js, c, h, java, etc.)

- Archivos comprimidos (gz, tar, zip, etc.)

- Rich Text (rtf)

- ePub

- Audio/Imagen/Video (extracción de metadatos)

- mbox

- Archivos ai (compatible con PDF)

Fess extrae cadenas de varios tipos de archivos desconocidos. Los archivos no enumerados anteriormente también se pueden rastrear y buscar. Si tiene archivos que desea verificar, envíe un pull request al `repositorio de datos de prueba para sistemas de búsqueda <https://github.com/codelibs/fess-testdata>`__.
