Política de Versionado
====================

Las versiones se proporcionan en el formato X.Y.Z.

-  X: Versión principal. La versión aumenta cuando se actualiza el esquema de Elasticsearch de Fess.

-  Y: Versión menor. La versión aumenta para adiciones de funciones o correcciones de errores que no actualizan el esquema de Elasticsearch de Fess.

-  Z:
   Versión micro. La versión aumenta para correcciones de errores críticos.
   (Incluso si no es crítico, si es necesario según las solicitudes de los usuarios. Se manejará caso por caso)
