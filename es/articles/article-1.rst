Creación de un Entorno de Búsqueda Empresarial con Fess ~ Edición de Introducción
===============================================

Introducción
========

Este artículo presenta la construcción de un servidor de búsqueda utilizando Fess.

Fess es un servidor de búsqueda de texto completo de código abierto basado en Java. Utiliza Elasticsearch como motor de búsqueda y Fess Crawler para rastrear documentos web y del sistema de archivos.

Consulte la `documentación completa <https://fess.codelibs.org/ja/articles/article-1.html>`__ para obtener instrucciones detalladas de instalación y configuración.
