=================
Búsqueda avanzada
=================

La pantalla de búsqueda avanzada permite realizar búsquedas más avanzadas combinando varias condiciones.

Cómo utilizar
-------------

Se puede acceder a la pantalla de búsqueda avanzada desde el enlace «Búsqueda avanzada» en las opciones de búsqueda de la pantalla de búsqueda.

|image0|

Cuando se abre desde la pantalla de resultados de búsqueda, las palabras clave de búsqueda ya ingresadas se trasladan al campo «Contiene todas las palabras». Ingrese las condiciones en cada campo y presione el botón de búsqueda en la parte inferior de la página. Los múltiples campos ingresados se combinan en una única condición de búsqueda.

Lista de elementos
------------------

Contiene todas las palabras
:::::::::::::::::::::::::::

Busca documentos que contengan todas las palabras ingresadas (búsqueda AND).
Se pueden especificar varias palabras separadas por espacios.

Coincidencia exacta incluyendo el orden de las palabras
:::::::::::::::::::::::::::::::::::::::::::::::::::::::::

Busca documentos que contengan exactamente la cadena de texto ingresada, incluido el orden de las palabras (búsqueda de frase).
Dado que toda la cadena, incluidos los espacios, se trata como una única frase, no se puede dividir en palabras individuales para la búsqueda.

Contiene alguna de las palabras
:::::::::::::::::::::::::::::::

Busca documentos que contengan alguna de las palabras ingresadas (búsqueda OR).
Se pueden especificar varias palabras separadas por espacios.

No contiene palabras
::::::::::::::::::::

Busca documentos que no contengan las palabras ingresadas.
Se pueden especificar varias palabras separadas por espacios, y todas las palabras especificadas se excluyen de los resultados de búsqueda.

Número de resultados mostrados
::::::::::::::::::::::::::::::

Especifica el número de resultados de búsqueda que se muestran por página. Puede elegir entre 10, 20, 30, 40, 50 o 100. Si no se especifica, se utiliza el número de resultados mostrados predeterminado (valor inicial 10).

Ordenar
:::::::

Especifica el criterio para ordenar los resultados de búsqueda. Puede elegir entre Por puntuación, Nombre de archivo, Fecha, Tamaño y Última modificación, cada uno en orden ascendente o descendente. La ordenación por Clics o Favoritos está disponible cuando la función correspondiente está habilitada.

Idioma preferido
::::::::::::::::

Especifica el idioma preferido para los resultados de búsqueda. Se pueden seleccionar varios idiomas. Si se selecciona «Todos los idiomas», no se prioriza ningún idioma específico.

Etiqueta
::::::::

Filtra la búsqueda por etiqueta. Se pueden seleccionar varias etiquetas. No se muestra si no hay etiquetas registradas o si no hay etiquetas que el usuario actual pueda ver.

Fecha de actualización
::::::::::::::::::::::

Filtra la búsqueda por la fecha y hora de actualización del documento. Puede elegir entre Cualquier fecha, Últimas 24 horas, Última semana, Último mes y Último año.

Formato de archivo
::::::::::::::::::

Filtra la búsqueda por el formato de archivo del documento. Puede elegir entre Todos los formatos, HTML, PDF, MS Word, MS Excel y MS PowerPoint.

Objetivo de búsqueda
::::::::::::::::::::

Especifica dónde buscar. Puede elegir entre Página completa, Título de la página y URL de la página. Si se selecciona la opción de título o URL, las palabras ingresadas se buscan únicamente en el título o en la URL respectivamente.

Sitio o dominio
:::::::::::::::

Filtra la búsqueda por el sitio o dominio ingresado. Especifíquelo cuando desee buscar únicamente documentos contenidos en un sitio o dominio específico.
