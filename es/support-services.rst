============
Soporte Comercial
============

Acerca del Soporte Comercial
====================

Fess es de código abierto proporcionado bajo la licencia Apache y puede usarse libremente sin costo tanto para uso personal como comercial.

`N2SM, Inc. <https://www.n2sm.net/>`__ proporciona lo siguiente:

- `N2 Search <https://www.n2sm.net/n2search.html>`__: Versión de producto comercial de Fess

- `Soporte de soluciones de búsqueda <https://www.n2sm.net/products/n2search/support.html>`__: Personalización y asistencia en la construcción de Fess y Elasticsearch, etc.

N2 Search
=========

`N2 Search <https://www.n2sm.net/n2search.html>`__ difiere de Fess en los siguientes puntos:

- Mejora del rendimiento de búsqueda

- Optimización del uso del disco

- Fortalecimiento de la seguridad de las funciones de administración

- Análisis morfológico compatible con nuevas palabras

Servicios de Soporte
================

Si necesita servicios como los siguientes, considere el `soporte comercial (soporte de soluciones de búsqueda) <https://www.n2sm.net/products/n2search/support.html>`__.

- Consultoría de implementación y operación

- Desarrollo de personalizaciones según requisitos individuales (el proyecto Fess responde a requisitos genéricos, pero no a requisitos considerados individuales)

- Manejo de rastreo de grandes volúmenes de documentos (proyectos que aplican a más de 500,000 documentos requieren conocimiento suficiente de Fess/Elasticsearch)

- Configuración de clustering

- Optimización de resultados de búsqueda para búsqueda empresarial/búsqueda interna corporativa

- Ajuste de servidores Fess/Elasticsearch

- Investigación y manejo de archivos que no se indexan (el proyecto Fess no proporciona soporte adicional cuando las bibliotecas dependientes no pueden leer, consulte el soporte comercial si es necesario)

- Construcción de entorno de sistema de búsqueda de texto completo de alta velocidad

- Mantenimiento de versiones específicas (el proyecto Fess solo realiza correcciones a la última versión)

- Capacitación

- Otros

AWS Marketplace
===============

Aunque tiene límites de funcionalidad como límites en el número de documentos, puede utilizar N2 Search en AWS Marketplace.
También se puede implementar fácilmente como motor de búsqueda de documentos para pequeñas y medianas empresas.
Consulte `N2 Search <https://aws.amazon.com/marketplace/pp/B014JFU5EW>`__.
