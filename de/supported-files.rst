================
Durchsuchbare Dateien
================

Unterstützte Dateiformate
==========================

Liste der Dateiformate, die Fess für Crawling und Suche unterstützt.

- Text (txt)

- XML (xml, xhtml, mm usw.)

- HTML (html, htm)

- MS Office (doc, xls, ppt, docx, xlsx, pptx usw.)

- PDF (pdf usw.)

- Quellcode (js, c, h, java usw.)

- Komprimierte Dateien (gz, tar, zip usw.)

- Rich Text Format (rtf)

- ePub

- Audio/Bild/Video (Metadaten-Extraktion)

- mbox

- ai-Dateien (PDF-kompatibel)

Fess extrahiert auch Textinhalte aus verschiedenen unbekannten Dateitypen. Auch nicht oben aufgeführte Dateien können gecrawlt und durchsucht werden. Wenn Sie ein bestimmtes Dateiformat überprüfen möchten, senden Sie bitte einen Pull Request an das `Test-Datenrepository für Suchsysteme <https://github.com/codelibs/fess-testdata>`__.
