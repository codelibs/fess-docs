======
使用方法
======

本页面说明 Fess 的基本使用方法。
如果尚未安装 Fess,请参考 :doc:`setup` 或 :doc:`quick-start`。

关于 Fess 的 UI
===================

Fess 提供以下 UI。

-  基于浏览器的搜索·搜索结果 UI(面向普通用户)
-  基于浏览器的管理 UI(面向管理员)

搜索界面(面向普通用户)
===========================

访问搜索界面
------------------

这是普通用户搜索 Fess 索引的文档的 UI。

访问 http://localhost:8080/ 后,会显示搜索词输入栏和搜索按钮。

基本搜索
----------

输入要搜索的单词并点击搜索按钮后,会显示搜索结果。

|基于浏览器的搜索结果显示|

搜索结果会显示以下信息:

- 标题
- URL
- 正文摘录(搜索关键词会高亮显示)
- 最后更新时间
- 文件大小(文档的情况)

高级搜索
--------

**AND 搜索**

用空格分隔输入多个关键词,会搜索包含所有关键词的文档。

例: ``Fess 搜索`` → 包含「Fess」和「搜索」两者的文档

**OR 搜索**

在关键词之间输入 ``OR``,会搜索包含任一关键词的文档。

例: ``Fess OR Elasticsearch`` → 包含「Fess」或「Elasticsearch」的文档

**NOT 搜索**

在要排除的关键词前加 ``-``,会搜索不包含该关键词的文档。

例: ``Fess -Elasticsearch`` → 包含「Fess」但不包含「Elasticsearch」的文档

**短语搜索**

用 ``""`` 包围关键词,会进行完全匹配的短语搜索。

例: ``"全文搜索"`` → 包含「全文搜索」这个词的文档

搜索选项
------------

搜索界面可使用以下选项:

- **标签搜索**: 仅搜索附加了特定标签的文档
- **期间指定**: 仅搜索在特定期间更新的文档
- **文件格式指定**: 仅搜索特定文件格式(PDF、Word 等)

管理界面(面向管理员)
====================

访问管理界面
------------------

这是管理员管理 Fess 的 UI。

访问 http://localhost:8080/admin/ 后,会显示登录界面。

默认管理员账号:

- **用户名**: ``admin``
- **密码**: ``admin``

.. warning::

   **关于安全性的重要注意事项**

   请务必更改默认密码。
   特别是在生产环境中,强烈建议在首次登录后立即更改密码。

.. note::

   管理 UI 不支持响应式网页设计。
   建议从 PC 浏览器访问。

主要管理功能
----------

登录后,可以访问以下设置·管理功能:

**爬虫设置**

- 网络爬取设置
- 文件系统爬取设置
- 数据存储爬取设置

**系统设置**

- 常规设置(时区、邮件设置等)
- 用户·角色管理
- 调度器设置
- 设计设置

**搜索设置**

- 标签管理
- 关键词相关度调整
- 同义词·词典管理

关于详细的管理方法,请参考用户指南。
