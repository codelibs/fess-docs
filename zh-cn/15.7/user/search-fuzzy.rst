模糊检索
======

模糊检索(Fuzzy检索)
=================

当您想搜索与搜索词不完全匹配的单词时,可以使用模糊检索。\ |Fess| 支持基于莱文斯坦距离的模糊检索(Fuzzy 检索)。这在因输入错误、表记差异或拼写不同等原因导致无法完全匹配,但仍希望检索到相关文档时非常有用。莱文斯坦距离(编辑距离)用于表示两个单词之间存在多大差异。

使用方法
------

在要应用模糊检索的搜索词后添加"~"。

例如,如果想对"Fess"这个单词进行模糊检索,通过在搜索表单中输入如下内容,可以搜索包含与"Fess"接近的词(如"Fes"等)的文档。

::

    Fess~

在"~"后添加 0~2 之间的数字,可以指定允许的编辑距离(即字符插入、删除、替换等操作的最大次数)。省略数字时,将采用默认的编辑距离。

::

    Fess~1
    Fess~2

指定"~2"时,可以搜索到与"Fess"相差最多两个字符的内容。
