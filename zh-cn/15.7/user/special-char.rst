======
特殊字符
======

以下字符在搜索词中作为特殊字符处理。当要将这些字符作为搜索字符使用时,需要进行转义。例如,搜索词中包含 URL 或文件路径里的"/"":",或程序代码中的"+"、"-"等符号时,如果直接搜索而不加转义,可能会得到意料之外的搜索结果。转义方法请参见下文。

::

    + - && || ! ( ) { } [ ] ^ " ~ * ? : \ /

   * - ``&&`` ``||``
     - AND 检索 / OR 检索
     - :doc:`search-and` / :doc:`search-or`

     - 范围指定检索(``[ ]`` 包含边界值,``{ }`` 不包含边界值)

     - 提升检索(加权检索)
     - :doc:`search-boost`

使用方法
------

使用 \ 进行转义,或用 " 将搜索词括起来输入。

::

    aaa\/bbb
    "aaa/bbb"

例如,要将"C++"作为搜索词进行检索时,可以按如下方式转义。

::

    C\+\+
    "C++"
