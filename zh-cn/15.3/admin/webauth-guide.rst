========
Web 认证
========

概述
====

本节介绍针对 Web 爬取需要 Web 认证时的配置方法。
|Fess| 支持 BASIC 认证、DIGEST 认证和 NTLM 认证的爬取。

管理方法
======

显示方法
------

要打开下图所示的 Web 认证配置列表页面,请点击左侧菜单中的 [爬虫 > Web 认证]。

|image0|

点击配置名称进行编辑。

创建配置
--------

要打开 Web 认证配置页面,请点击新建按钮。

|image1|

配置项
------

主机名
::::::

指定需要认证的站点的主机名。
如果省略,将在指定的 Web 爬取配置中对任意主机名应用。

端口
:::::

指定需要认证的站点的端口。
如果要对所有端口应用,请指定 -1。
在这种情况下,将在指定的 Web 爬取配置中对任意端口应用。

领域
:::::

指定需要认证的站点的领域名。
如果省略,将在指定的 Web 爬取配置中对任意领域名应用。

方案
::::::

选择认证方法。
可以使用 BASIC 认证、DIGEST 认证、NTLM 认证或 FORM 认证。

用户名
::::::::

指定登录认证站点的用户名。

密码
::::::::

指定登录认证站点的密码。

参数
:::::::::

在需要设置登录认证站点所需的配置值时进行设置。
对于 NTLM 认证,可以设置 workstation 和 domain 的值。
设置时请按以下方式描述。

::

    workstation=HOGE
    domain=FUGA

Web 配置
::::::::

选择要应用上述认证设置的 Web 配置名称。
需要事先注册 Web 爬取配置。

删除配置
--------

点击列表页面中的配置名称,然后点击删除按钮,将显示确认画面。
点击删除按钮将删除配置。
