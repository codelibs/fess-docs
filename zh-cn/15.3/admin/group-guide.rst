======
组
======

概述
====

可以管理用户所属的组。
在与 LDAP 集成等情况下可以使用。

管理方法
======

显示方法
------

要打开下图所示的组设置列表页面，请点击左侧菜单中的 [用户 > 组]。

|image0|

点击设置名称可进行编辑。

创建设置
--------

要打开组设置页面，请点击新建按钮。

|image1|

配置项
------

名称
::::

组名称。

删除方法
--------

点击列表页面的设置名称，然后点击删除按钮，将显示确认画面。
点击删除按钮将删除该设置。
