========
存储
========

概述
====

存储页面可以管理 Amazon S3 兼容的对象存储 MinIO 上的文件。

管理方法
======

对象存储服务器配置
---------------------------

从[系统 > 常规]打开存储配置,设置以下项:

- 端点: 存储服务器的端点 URL
- 访问密钥: 存储服务器的访问密钥
- 私密密钥: 存储服务器的私密密钥
- 存储桶: 要管理的存储桶名称


显示方法
------

要打开下图所示的对象列表页面,请单击左侧菜单中的[系统 > 存储]。

|image0|


名称

对象的文件名

对象的大小

:::::::::

对象的最后更新时间

下载
---------

单击"下载"按钮,即可下载对象。


删除
----

单击"删除"按钮,即可删除对象。


上传
---------

单击右上角的"文件上传"按钮,即可打开文件上传窗口。


创建文件夹
------------

单击路径显示右侧的"创建文件夹"按钮,即可打开创建文件夹窗口。请注意,无法创建空文件夹。
