==============
通配符检索
==============

可以在搜索词中使用单字符或多字符通配符。? 可以指定为单字符通配符,* 可以指定为多字符通配符。通配符的使用对象是单词。不能对句子进行通配符检索。

使用方法
------

使用单字符通配符时,使用 ? ,如下所示。

::

    te?t

上述情况将作为单字符通配符处理,如 text 或 test 等。

使用多字符通配符时,使用 * ,如下所示。

::

    test*

上述情况将作为多字符通配符处理,如 test、tests 或 tester 等。此外,

::

    te*t

也可以在搜索词中这样使用。

使用条件
------

通配符针对索引中注册的字符串使用。
因此,如果使用 bi-gram 等创建索引,日语会被当作无意义的固定字符串长度处理,日语通配符将无法达到预期效果。
在日语中使用通配符时,请在使用形态素分析的字段中使用。
