=========
文件认证
=========

概述
====

本节介绍在爬取文件时需要进行文件认证的配置方法。
|Fess| 支持对 FTP 或 Windows 共享文件夹进行认证。

管理方法
======

显示方法
------

要打开下图所示的文件认证设置列表页面，请点击左侧菜单中的 [爬虫 > 文件认证]。

|image0|

点击设置名称可进行编辑。

创建设置
--------

要打开文件认证设置页面，请点击新建按钮。

|image1|

配置项
------

主机名
::::::

指定需要认证的站点的主机名。

端口
::::

指定需要认证的站点的端口。

协议
::::::

选择认证方法。
可以使用 FTP 或 SAMBA（Windows 共享文件夹认证）。

用户名
::::::::

指定登录认证站点的用户名。

密码
::::::::

指定登录认证站点的密码。

参数
:::::::::

在需要设置登录认证站点所需的配置值时使用。对于 SAMBA，可以设置 domain 的值。设置时请按如下方式描述。

::

    domain=FUGA

文件爬取设置
:::::::::::::::

指定使用此认证设置的爬取配置。

删除设置
--------

点击列表页面的设置名称，然后点击删除按钮，将显示确认画面。
点击删除按钮将删除该设置。
