========
角色检索
========

|Fess| 提供用户管理功能，登录用户可以根据自己所属的角色进行检索。\ |Fess| 管理的用户在登录后，既可以使用角色检索，也可以更改自己的密码。

在角色检索中，系统会将内容中设置的权限（角色、组、用户）与检索用户所持有的权限进行核对，仅在检索结果中显示允许访问的内容。有关角色和用户的创建方法，以及为内容分配权限的方法等基于角色的检索配置，请参阅 :doc:`../config/security-role`。


检索方法
------

在设置角色并完成爬取和索引后，检索结果可以只显示给拥有该角色的用户。
如果用户处于登录状态，将根据该用户所属的角色或组进行检索。
如果未登录，则以 guest 用户身份进行检索，仅显示面向 guest 公开的内容。

登录
------

点击检索界面上部显示的登录链接，将显示登录界面。输入用户名和密码进行登录后，将返回检索界面，此后的检索将基于该用户所属的角色进行。

.. note::
    如果与单点登录或 LDAP 集成，则按各自的认证方式登录。此外，登录链接是否显示可以通过设置进行更改。

更改密码
------------

登录后，点击检索界面上部显示的用户名，将显示菜单。

|image0|

在菜单中点击「更改密码」，将显示更改密码界面。

|image1|

输入当前密码、新密码以及用于确认的新密码（再次输入），点击更新按钮即可更新密码。
更改密码后，点击返回按钮即可返回检索界面。

.. note::
    「更改密码」菜单仅显示给 |Fess| 管理的用户（以及被允许编辑的 LDAP 用户）。通过单点登录认证的用户不会显示该菜单。
    新密码可能会应用长度、可使用字符类型等密码策略。

注销
--------

在登录状态下，点击检索界面上部显示的用户名，从菜单中选择「注销」即可退出登录。
如果用户拥有管理员权限，也可以从同一菜单中选择「管理」，进入管理界面。
