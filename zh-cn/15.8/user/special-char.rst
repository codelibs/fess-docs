======
特殊字符
======

以下字符在搜索查询语法中具有特殊含义，因此在搜索词中被视为特殊字符。如果要将这些字符作为普通搜索字符使用，需要对其进行转义。

::

    + - && || ! ( ) { } [ ] ^ " ~ * ? : \ /

这些字符用于调用必需词与排除词（``+`` ``-``）、布尔运算符（``&&`` ``||`` ``!``）、分组（``( )``）、范围搜索（``[ ]`` ``{ }``）、权重搜索（``^``）、短语搜索（``"``）、模糊搜索（``~``）、通配符搜索（``*`` ``?``）以及字段搜索（``:``）等搜索功能。


使用方法
------

如果要将特殊字符作为普通搜索字符处理，请使用以下任一方法。

* 在字符前紧跟加上 ``\`` 进行转义。紧随其后的单个字符将被视为普通字符，而不是查询语法。
* 用 ``"`` 将搜索词括起来。括起来的字符串将被视为短语搜索，其中的特殊字符不会被解释为查询语法。但需要注意的是，由于会变成短语搜索，因此无法使用通配符搜索（``*`` ``?``）等功能。

::

    aaa\/bbb
    "aaa/bbb"
