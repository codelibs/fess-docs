标签指定检索(分类检索)
=======================

通过为搜索对象的文档添加用于分类的标签信息,可以在搜索时指定标签进行筛选检索。使用标签后,例如可以按部门、网站、文档类型等限定搜索范围。

在管理界面中预先注册标签后,就可以在搜索界面中使用标签进行筛选搜索。可用的标签可以在搜索时从下拉菜单中选择多个,如果选择了多个标签,则会检索出被赋予其中任意一个标签的文档。如果没有注册标签,则不会显示标签下拉框。

.. note::
    由于可以为标签设置权限,因此下拉菜单中只会显示允许检索用户访问的标签。此外,根据虚拟主机或语言区域(语言)的不同,显示的标签也可能会有所变化。因此,即使已注册了标签,对于某些用户来说,该标签也可能不会显示在下拉菜单中。

标签是通过 URL 路径的正则表达式来指定要附加标签的对象进行定义的。关于标签的注册方法和设置项,请参考 :doc:`标签管理指南 <../admin/labeltype-guide>` 。

使用方法
------

搜索时可以选择标签信息。单击选项按钮,在显示的搜索选项中可以选择标签信息。

|image0|

通过设置标签并创建索引,可以按照设置了标签的文档进行搜索。不指定标签的搜索与通常的全文检索相同。

为文档赋予标签是在爬取并创建索引时,通过将文档的 URL 与标签中设置的路径进行比对来完成的。因此,如果新增或更改了标签的定义(目标路径或排除路径),其内容不会自动反映到已经建立索引的文档中。要使更改生效,请重新爬取目标文档,或执行调度器中注册的"Label Updater"作业来更新索引。
