=========
提升检索
=========

提升检索(加权检索)
====================

当您想优先处理特定搜索词时,可以使用提升检索。通过使用提升检索,可以根据搜索词的重要程度进行检索。

使用方法
------

要使用提升检索,需要在搜索词后以"^提升值"的格式指定提升值(加权值)。搜索词与提升值之间不加空格。

例如,在搜索"苹果 橘子"时,如果想优先显示"苹果",可以在搜索表单中输入如下内容:

::

    苹果^100 橘子

该示例仅对"苹果"应用了提升值,对"橘子"不应用提升值。

提升值
------

提升值需指定数值。除了整数外,还可以指定如 ``2.5`` 这样的小数。

- 指定大于 1 的值时,该搜索词的权重会增加。
- 指定大于 0 且小于 1 的值(例如 ``0.5``)时,该搜索词的权重会降低。

提升值并非绝对地决定搜索结果的得分,而是用于调整与其他搜索词之间的相对权重。

与其他检索语法的组合
--------------------

提升检索可以与其他检索语法组合使用。例如,也可以在指定了字段的搜索词上附加提升值。

::

    title:苹果^100 橘子

该示例中,title 字段中包含"苹果"的文档会被优先显示。
