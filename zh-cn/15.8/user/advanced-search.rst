========
高级检索
========

在高级检索界面可以通过组合多个条件进行更高级的检索。

使用方法
--------

可以从搜索界面的搜索选项中的“高级检索”链接访问高级检索界面。从搜索结果界面打开时,已输入的检索关键字会带入“包含所有单词”字段中。

|image0|

在各个字段中输入条件,然后单击页面下部的搜索按钮即可进行检索。输入的多个字段会组合成一个检索条件。

项目列表
--------

包含所有单词
::::::::::::

可以检索包含输入的所有单词的文档（AND 检索）。多个单词以空格分隔指定。

包括词序的完全匹配
::::::::::::::::::

可以检索完全包含输入字符串（含词序）的文档（短语检索）。由于包含空格在内的整个字符串会作为一个短语处理,因此无法拆分为各个单词进行检索。

包含任一单词
::::::::::::

可以检索包含输入的任一单词的文档（OR 检索）。多个单词以空格分隔指定。

不包含的单词
::::::::::::

可以检索不包含输入单词的文档。多个单词以空格分隔指定,指定的所有单词都会从检索结果中排除。

显示数量
::::::::

可以指定每页显示的搜索结果数量。可以从 10、20、30、40、50、100 中选择。如果不指定,则使用默认显示数量（初始值为 10）。

排序
::::

可以指定搜索结果的排序基准。可以从按分数、文件名、日期、大小、上次修改时间中选择,每种都可选择升序或降序。当启用相应功能时,还可以按点击数或收藏数排序。

优先语言
::::::::

可以指定搜索结果中优先的语言。可以选择多种语言。如果选择“所有语言”,则不优先特定语言。

标签
::::

可以通过标签筛选检索。可以选择多个标签。如果没有注册标签,或者没有当前用户可查看的标签,则不会显示。

更新日期
::::::::

可以通过文档的更新日期时间筛选检索。可以从不限、过去24小时、过去1周、过去1个月、过去1年中选择。

文件格式
::::::::

可以通过文档的文件格式筛选检索。可以从所有类型、HTML、PDF、MS Word、MS Excel、MS PowerPoint 中选择。

检索对象
::::::::

可以指定检索的位置。可以从整个页面、页面标题、页面URL中选择。如果选择页面标题或页面URL选项,则仅分别在标题或 URL 中检索输入的单词。

网站或域名
::::::::::

可以通过输入的网站或域名筛选检索。当只想检索特定网站或域名中包含的文档时指定。
