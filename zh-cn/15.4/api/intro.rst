目标读者
======

本文档面向使用 |Fess| API功能的开发人员。
阅读本文档需要具备以下知识：

- REST API基本概念
- JSON格式数据
- HTTP协议基础知识

阅读本文档前
============

本文档介绍了 |Fess| 15.4 中API的使用方法。

API使用前提条件
==============

使用 |Fess| API之前，需要满足以下条件：

- 已安装 |Fess| 15.4或更高版本，并正常启动
- 在管理界面中启用各API所需的设置（请参阅各API文档）
- 可通过网络访问API端点
