========
虚拟主机
========

关于虚拟主机
==============

可以根据访问 |Fess| 时的主机名(URL的主机部分)来区分搜索结果。
搜索结果在各自的 JSP 中显示,因此也可以更改设计。

系统配置
----------

在 :doc:`管理员指南 > 常规配置 <../admin/general-guide>` 中配置"虚拟主机"。在爬取配置中指定这里配置的虚拟主机名。

**格式**

::

   Host:主机名[:端口号]=虚拟主机名

.. list-table::

   * - *主机名*
     - 系统上可以名称解析(DNS)的主机名或IP地址
   * - *端口号*
     - 可选。默认为80。
   * - *虚拟主机名*
     - 在爬取配置中指定的虚拟主机名

**示例**

::

   Host:abc.example.com:8080=host1
   Host:192.168.1.123:8080=host2

进行配置后,将在 ``WEB-INF/view/虚拟主机名`` 中生成搜索页面的 JSP。
通过编辑这些文件,可以为每个虚拟主机更改页面设计。


爬取配置
---------

在 Web 爬取配置、文件爬取配置或数据存储爬取配置中指定"虚拟主机"。
"虚拟主机"请指定系统配置中配置的虚拟主机名之一。

**示例**

.. list-table::

   * - *虚拟主机*
     - ``host1``
