============
商业支持
============

关于商业支持
====================

Fess 是采用 Apache 许可证提供的开源软件，个人和商业用途均可免费使用。

`N2SM, Inc. <https://www.n2sm.net/>`__ 提供以下服务：

- `N2 Search <https://www.n2sm.net/n2search.html>`__: Fess 的商业产品版

- `搜索解决方案支持 <https://www.n2sm.net/products/n2search/support.html>`__: Fess 和 Elasticsearch 的定制及构建支持等

N2 Search
=========

`N2 Search <https://www.n2sm.net/n2search.html>`__ 与 Fess 相比有以下不同点：

- 搜索性能提升

- 磁盘使用量优化

- 管理功能的安全性强化

- 支持新词的形态素分析

支持服务
================

如需以下服务，请考虑 `商业支持（搜索解决方案支持） <https://www.n2sm.net/products/n2search/support.html>`__。

- 导入和运营咨询

- 个别需求的定制开发（Fess 项目会处理通用需求，但不处理被认为是个别需求的情况）

- 大量文档的爬取支持（对于超过50万文档的项目应用需要充分了解 Fess/Elasticsearch）

- 集群配置

- 企业内搜索、内网搜索的搜索结果优化

- Fess/Elasticsearch 服务器调优

- 无法索引的文件的调查和处理（Fess 项目在依赖库无法读取的情况下不会提供进一步支持，如有需要请咨询商业支持）

- 高速全文搜索系统环境的构建

- 特定版本的维护（Fess 项目只对最新版本进行修复）

- 培训

- 其他

AWS Marketplace
===============

虽然有文档数量上限等功能限制，但可以在 AWS Marketplace 上使用 N2 Search。
作为面向中小企业的文档搜索引擎也可以轻松导入。
请查看 `N2 Search <https://aws.amazon.com/marketplace/pp/B014JFU5EW>`__。
