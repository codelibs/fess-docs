========
插件
========

概述
====

插件配置页面用于管理插件。

管理方法
======

显示方法
------

要打开下图所示的已安装插件列表页面,请单击左侧菜单中的[系统 > 插件]。

|image0|

要卸载,请单击"删除"按钮。

安装
---------

要安装新插件,请单击"安装"按钮。

|image1|

在下拉菜单中选择要安装的插件,然后单击"安装"按钮,即可开始安装。
