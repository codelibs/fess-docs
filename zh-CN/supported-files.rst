支持的文件格式
============================

以下是 Fess 已确认可以爬取和搜索的文件格式列表。

- 文本 (txt)

- XML (xml, xhtml, mm等)

- HTML (html, htm)

- MS Office (doc, xls, ppt, docx, xlsx, pptx等)

- PDF (pdf等)

- 源代码 (js, c, h, java等)

- 压缩文件 (gz, tar, zip等)

- 富文本 (rtf)

- ePub

- 音频/图像/视频（元数据提取）

- mbox

- ai文件（PDF兼容）

Fess 会从各种未知类型的文件中提取字符串。上述未列出的文件也可以爬取和搜索。如果您想确认某个文件，请向\ `搜索系统测试数据仓库 <https://github.com/codelibs/fess-testdata>`__\ 提交拉取请求。
