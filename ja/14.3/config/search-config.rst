==================
検索関連の設定
==================

以下で説明している設定は fess_config.properties で指定します。
変更後は |Fess| の再起動が必要です。

ファジー検索
======================

4文字以上はファジー検索が適用されており、1文字違いもヒットします。
この設定を無効にする場合は、 -1 を指定します。
::

query.boost.fuzzy.min.length=-1

検索時のタイムアウト値
======================

検索時のタイムアウト値を指定することができます。
初期値は10秒です。

::

query.timeout=10000
