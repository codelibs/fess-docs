============
商用サポート
============

商用サポートについて
====================

Fess は Apache ライセンスで提供されるオープンソースで、個人や商用向けでも無料でご自由にご利用いただけます。

`N2SM, Inc. <https://www.n2sm.net/>`__ は以下を提供しています。

- `N2 Search <https://www.n2sm.net/n2search.html>`__: Fessの商用製品版

- `検索ソリューションのサポート <https://www.n2sm.net/products/n2search/support.html>`__: Fess や OpenSearch のカスタマイズおよび構築支援等

N2 Search
=========

`N2 Search <https://www.n2sm.net/n2search.html>`__ は Fess と比べて以下の点が異なります。

- 検索パフォーマンスの向上

- ディスク使用量の最適化

- 管理機能のセキュリティの強化

- 新語対応の形態素解析

サポートサービス
================

以下のようなサービスが必要な場合は `商用サポート(検索ソリューションのサポート) <https://www.n2sm.net/products/n2search/support.html>`__ をご検討ください。

- 導入・運用コンサルティング

- 個別要件によるカスタマイズ開発 (Fess プロジェクトでは汎用的な要件には対応しますが、個別と思われる要件は対応しません)

- 大量ドキュメントのクロール対応 (50万ドキュメントを超えるような案件への適用では Fess/OpenSearchに対する十分な知識が必要になります)

- クラスタリング構成

- 自社の企業内検索・社内検索に検索結果の最適化

- Fess/OpenSearch サーバのチューニング

- インデックス対象とならないファイルの調査・対応 (Fess プロジェクトでは依存するライブラリが読み取れない場合はそれ以上のことはサポートしないため、必要であれば商用サポートでご相談ください)

- 高速全文検索システム環境の構築

- 特定のバージョンの保守 (Fess プロジェクトでは最新バージョンへの修正しか行いません)

- トレーニング

- その他

お問い合わせ
==========

商用サポートに関するお問い合わせは、以下の方法でご連絡ください。

- **Web フォーム**: `N2SM お問い合わせフォーム <https://www.n2sm.net/contact.html>`__
- **会社情報**: `N2SM, Inc. <https://www.n2sm.net/>`__

お問い合わせ時にご準備いただきたい情報：

1. ご利用中（または検討中）の Fess のバージョン
2. 想定されるドキュメント数と利用規模
3. 必要なサポート内容（導入支援、カスタマイズ、運用サポート等）
4. ご希望の導入時期

よくある質問
==========

Q: オープンソース版と商用版の違いは何ですか？
------------------------------------------

A: オープンソース版（Fess）は無料でご利用いただけますが、技術サポートはコミュニティベースとなります。
商用版（N2 Search）は、検索パフォーマンスの向上、セキュリティ機能の強化、およびプロフェッショナルサポートが含まれています。

Q: 導入前に試用することはできますか？
----------------------------------

A: はい。オープンソース版の Fess は `ダウンロードページ <downloads.html>`__ からすぐにダウンロードして試用できます。
N2 Search については、お問い合わせフォームよりトライアルのご相談をお願いします。

Q: 既存の Fess から N2 Search への移行は可能ですか？
-------------------------------------------------

A: はい、移行可能です。詳細な移行手順については、商用サポートにお問い合わせください。
インデックスデータの移行方法や設定の引き継ぎについてご案内いたします。

Q: サポート契約は年間契約ですか？
------------------------------

A: サポート契約の期間については、お客様のご要望に応じて柔軟に対応いたします。
詳細は `N2SM お問い合わせフォーム <https://www.n2sm.net/contact.html>`__ よりご相談ください。

Q: クラウド環境での利用は可能ですか？
----------------------------------

A: はい、AWS、Azure、GCP などの主要なクラウドプラットフォームでご利用いただけます。
Docker を使用した環境構築については、`Docker インストールガイド <15.5/install/install-docker.html>`__ を参照してください。
