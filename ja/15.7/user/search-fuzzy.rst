======
曖昧検索
======

曖昧検索(ファジー検索)
=================

検索語に完全一致しない単語も検索したい場合には、あいまい検索(ファジー検索)が利用できます。
タイプミスや表記ゆれ、つづり違いなどで完全一致しないドキュメントも拾いたい場合に有効です。
\ |Fess| は、2つの単語がどれだけ異なるかを表す編集距離(レーベンシュタイン距離)に基づいたあいまい検索(ファジー検索)に対応しています。

利用方法
------

曖昧検索を適用したい検索語の後に「~」を付加します。

たとえば、「Fess」という単語を曖昧検索したい場合、以下のように検索フォームに入力することで、「Fess」に近い語 (「Fes」など) を含むドキュメントを検索できます。

::

    Fess~

あいまいさの度合いは、``~`` の後ろに 0〜2 の数値を付けて指定できます。
数値は許容する編集距離(文字の挿入・削除・置換の最大回数)の上限を表し、省略した場合は既定の編集距離が適用されます。

::

    Fess~1
    Fess~2

``~2`` を指定すると、``Fess`` から2文字までの違いを許容して検索します。
