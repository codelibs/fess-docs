======
特殊文字
======

Fess の検索では、以下の記号は検索構文上の特殊文字（予約語）として扱われます。
これらの記号を通常の検索文字として検索したい場合は、エスケープが必要です。
たとえば URL やファイルパスに含まれる「/」「:」、プログラム中の「+」「-」などをそのまま検索すると、
意図しない検索結果になることがあります。エスケープの方法は以下を参照してください。

::

    + - && || ! ( ) { } [ ] ^ " ~ * ? : \ /


特殊文字の一覧と意味
--------------

.. list-table::
   :header-rows: 1

   * - 特殊文字
     - 意味・役割
     - 関連ページ
   * - ``+`` ``-``
     - 必須語・除外語の指定(AND 検索 / NOT 検索)
     - :doc:`search-and` / :doc:`search-not`
   * - ``&&`` ``||``
     - AND 検索 / OR 検索
     - :doc:`search-and` / :doc:`search-or`
   * - ``!``
     - NOT 検索(除外検索)
     - :doc:`search-not`
   * - ``( )``
     - 検索条件のグループ化
     - :doc:`advanced-search`
   * - ``[ ]`` ``{ }``
     - 範囲指定検索(``[ ]`` は境界を含む、``{ }`` は境界を含まない)
     - :doc:`search-range`
   * - ``^``
     - ブースト検索(重み付け)
     - :doc:`search-boost`
   * - ``"``
     - フレーズ検索(囲んだ範囲を1語句として扱う。エスケープの代わりにも使える)
     - :doc:`advanced-search`
   * - ``~``
     - あいまい検索(ファジー検索)
     - :doc:`search-fuzzy`
   * - ``*`` ``?``
     - ワイルドカード検索
     - :doc:`search-wildcard`
   * - ``:``
     - 検索フィールドの指定
     - :doc:`search-field`
   * - ``\``
     - エスケープ文字
     - (本ページ)
   * - ``/``
     - スラッシュ(URL などに含まれる場合はエスケープが必要)
     - :doc:`search-field`


利用方法
------

\ でエスケープするか、" で検索語を囲んだ検索語を入力します。

::

    aaa\/bbb
    "aaa/bbb"

たとえば「C++」を検索語として扱いたい場合は、以下のようにエスケープします。

::

    C\+\+
    "C++"
