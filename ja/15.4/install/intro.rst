======
はじめに
======

本ドキュメントについて
==================

このインストールガイドは、 |Fess| のインストール、起動、アップグレード、およびアンインストールの手順を説明します。
ご利用の環境に応じて、適切なインストール方法を選択してください。

対象読者
========

このドキュメントは、 |Fess| のインストール作業およびアンインストール作業を担当するユーザーを対象にしています。

お読みになる前に
==============

このドキュメントでは、 |Fess| のインストール方法を示しています。以下の知識が必要になります：

- サーバー管理の基礎知識
- Linux または Windows のコマンドライン操作
- ネットワークおよびファイアウォールの基本設定
- Docker の基礎知識 (Docker版をご利用の場合)

ドキュメント構成
==============

このインストールガイドは、以下のセクションで構成されています：

- **システム要件**: |Fess| を実行するために必要なハードウェアおよびソフトウェア要件
- **インストール**: プラットフォーム別のインストール手順

  - Linux (TAR.GZ/RPM/DEB) 向けインストール
  - Windows (ZIP) 向けインストール
  - Docker 向けインストール

- **起動および停止**: |Fess| サーバーの起動、停止、および初回セットアップ
- **アップグレード**: 以前のバージョンから最新版へのアップグレード手順
- **アンインストール**: |Fess| の完全なアンインストール手順
- **セキュリティ設定**: 本番環境で推奨されるセキュリティ設定
- **トラブルシューティング**: よくある問題と解決方法
