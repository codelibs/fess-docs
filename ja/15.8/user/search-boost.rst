=========
ブースト検索
=========

ブースト検索(重み付け検索)
====================

検索語のうち、特定の検索語を優先したい場合にはブースト検索を利用します。ブースト検索を利用することで、検索語の重要度に応じた検索が可能になります。

利用方法
------

ブースト検索を利用するためには、検索語の直後に「^ブースト値」という形式でブースト値 (重み付け値) を指定します。検索語とブースト値の間にはスペースを入れません。

たとえば、「りんご みかん」を検索する際に「りんご」をより優先したい場合は、以下のように検索フォームに入力します。

::

    りんご^100 みかん

この例では「りんご」にのみブースト値が適用され、「みかん」には適用されません。

ブースト値
--------

ブースト値には数値を指定します。整数だけでなく、``2.5`` のような小数も指定できます。

- 1 より大きい値を指定すると、その検索語の重みが増加します。
- 0 より大きく 1 未満の値 (例: ``0.5``) を指定すると、その検索語の重みが減少します。

ブースト値は検索結果のスコアを絶対的に決めるものではなく、他の検索語との相対的な重み付けを調整するものです。

他の検索構文との組み合わせ
--------------------

ブースト検索は他の検索構文と組み合わせて利用できます。たとえば、フィールドを指定した検索語にブースト値を付加することもできます。

::

    title:りんご^100 みかん

この例では、title フィールドに「りんご」が含まれるドキュメントが優先されます。
