はじめに
======

対象読者
======

このドキュメントは |Fess| のAPI機能を利用する開発者を対象にしています。
以下の知識を前提としています。

- REST APIの基本概念
- JSON形式のデータフォーマット
- HTTPプロトコルの基礎知識

お読みになる前に
============

このドキュメントでは、 |Fess| 15.3 でのAPIの使用方法を示しています。

API利用の前提条件
==============

|Fess| のAPIを利用する前に、以下の設定が必要です。

- |Fess| 15.3以降がインストールされ、正常に起動していること
- 管理画面からAPIごとに必要な設定を有効にすること（各APIドキュメントを参照）
- ネットワーク経由でAPIエンドポイントにアクセス可能であること
