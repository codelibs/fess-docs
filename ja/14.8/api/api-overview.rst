|Fess| の提供するAPI
====================

このドキュメントでは、 |Fess| が提供するAPIについて説明します。
APIを利用することで、既存のウェブシステムなどでも、|Fess| を検索サーバーとして使うことができます。

APIの種類
=========

.. TODO: favorite, favorites

|Fess| はいくつかのAPIを提供しています。

.. tabularcolumns:: |p{3cm}|p{12cm}|
.. list-table::

   * - search
     - 検索結果を取得するAPI。
   * - popularword
     - 人気ワードを取得するAPI。
   * - label
     - 作成済みのラベルのリストを取得するAPI。
   * - ping
     - サーバーの状態を取得するAPI。
   * - suggest
     - サジェストワードを取得するAPI。
